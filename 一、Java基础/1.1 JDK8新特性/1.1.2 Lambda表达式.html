1.1.2 Lambda表达式

一、匿名内部类

能够使用Lambda的依据是必须有相应的函数接口（函数接口，是指内部只有一个抽象方法的接口）接口名和函数名都一同省掉了。

1、无参函数的简写

JDK7代码

			new Thread(new Runnable() {
            @Override
            public void run() {
                System.out.println("Thread run()");
            }
        }).start();

JDK8简写后代码

new Thread(()-> System.out.println("Thread run")).start();//单行
//多行        
new Thread(()-> {
            System.out.println("Thread run...");
            System.out.println("Thread run...");
        }).start();

2、带参数的简写

JDK7代码

        List<String> list= Arrays.asList("I", "love", "you", "too");
        Collections.sort(list, new Comparator<String>() {
            @Override
            public int compare(String o1, String o2) {
                if(o1 == null)
                    return -1;
                if(o2 == null)
                    return 1;
                return o1.length()-o2.length();
            }
        });

JDK8简写后代码

        List<String> list= Arrays.asList("I", "love", "you", "too");
        Collections.sort(list,(o1,o2)->{
            if(o1 == null)
                return -1;
            if(o2 == null)
                return 1;
            return o1.length()-o2.length();
        });

二、集合

1、forEach

JDK7代码

// 使用曾强for循环迭代
ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
for(String str : list){
    if(str.length()>3)
        System.out.println(str);
}

JDK8代码

        ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
        list.forEach(s->{
            if(s.length()>3)
                System.out.println(s);
        });

2、removeIf

删除容器中所有满足filter指定条件的元素

JDK7代码

// 使用迭代器删除列表元素
ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
Iterator<String> it = list.iterator();
while(it.hasNext()){
    if(it.next().length()>3) // 删除长度大于3的元素
        it.remove();
}

JDK8代码

        ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
        list.removeIf(s->s.length()>3);

3、replaceAll

对每个元素执行operator指定的操作，并用操作结果来替换原来的元素。

JDK7代码

// 使用下标实现元素替换
ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
for(int i=0; i<list.size(); i++){
    String str = list.get(i);
    if(str.length()>3)
        list.set(i, str.toUpperCase());
}

JDK8代码

        ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
        list.replaceAll(s->{
            if (s.length()>3){
                s=s.toUpperCase();
            }
            return s;
        });

4、sort

JDK7代码

ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
Collections.sort(list, new Comparator<String>(){
    @Override
    public int compare(String str1, String str2){
        return str1.length()-str2.length();
    }
});

JDK8代码

        ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
        list.sort((s1,s2)-> {
            return s1.length()-s2.length();
        });
//或者
ArrayList<String> list = new ArrayList<>(Arrays.asList("I", "love", "you", "too"));
list.sort((s1,s2)-> s1.length()-s2.length());

三、Map

1、forEach

JDK7代码

// Java7以及之前迭代Map
HashMap<Integer, String> map = new HashMap<>();
map.put(1, "one");
map.put(2, "two");
map.put(3, "three");
for(Map.Entry<Integer, String> entry : map.entrySet()){
    System.out.println(entry.getKey() + "=" + entry.getValue());
}

JKD8代码

        HashMap<Integer, String> map = new HashMap<>();
        map.put(1, "one");
        map.put(2, "two");
        map.put(3, "three");
        map.forEach((k,v)-> System.out.println(k+"="+v));

2、getOrDefault

按照给定的key查询Map中对应的value，如果没有找到则返回defaultValue

需求；假设有一个数字到对应英文单词的Map，输出4对应的英文单词，如果不存在则输出NoValue

// 查询Map中指定的值，不存在时使用默认值
HashMap<Integer, String> map = new HashMap<>();
map.put(1, "one");
map.put(2, "two");
map.put(3, "three");
// Java7以及之前做法
if(map.containsKey(4)){ // 1
    System.out.println(map.get(4));
}else{
    System.out.println("NoValue");
}
// Java8使用Map.getOrDefault()
System.out.println(map.getOrDefault(4, "NoValue")); // 2

3、replaceAll

JDK7

// Java7以及之前替换所有Map中所有映射关系
HashMap<Integer, String> map = new HashMap<>();
map.put(1, "one");
map.put(2, "two");
map.put(3, "three");
for(Map.Entry<Integer, String> entry : map.entrySet()){
    entry.setValue(entry.getValue().toUpperCase());
}

JDK8

        HashMap<Integer, String> map = new HashMap<>();
        map.put(1, "one");
        map.put(2, "two");
        map.put(3, "three");
        map.replaceAll((k,v)->v.toUpperCase());

四、Stream API 1

1、Stream方法的使用

1.1 forEach

遍历

// 使用Stream.forEach()迭代
        Stream<String> stream = Stream.of("I", "love", "you", "too");
        stream.forEach(s-> System.out.println(s));

1.2 filter

过滤

// 保留长度等于3的字符串并遍历输出
        Stream<String> stream= Stream.of("I", "love", "you", "too");
        stream.filter(s->s.length()>3)
                .forEach(s-> System.out.println(s));

1.3 distinct

去重

        Stream<String> stream= Stream.of("I", "love", "you", "too", "too");
        stream.distinct().forEach(s-> System.out.println(s));

1.4 sorted

排序

        Stream<String> stream= Stream.of("I", "love", "you", "too");
        stream.sorted((s1,s2)-> s1.length()-s2.length())
        .forEach(s-> System.out.println(s));

1.5 map

映射。作用是返回一个对当前所有元素执行执行mapper之后的结果组成的Stream

        Stream<String> stream= Stream.of("I", "love", "you", "too");
        stream.map(m->m.toUpperCase())
                .forEach(s-> System.out.println(s));

1.6 flatMap

俗的讲flatMap()的作用就相当于把原stream中的所有元素都”摊平”之后组成的Stream，转换前后元素的个数和类型都可能会改变。

        Stream<List<Integer>> stream = Stream.of(Arrays.asList(1,2), Arrays.asList(3, 4, 5));
        stream.flatMap(list -> list.stream())
                .forEach(i -> System.out.println(i));

2、collect

// 将Stream转换成List或Set
Stream<String> stream = Stream.of("I", "love", "you", "too");
List<String> list = stream.collect(Collectors.toList()); // (1)
Set<String> set = stream.collect(Collectors.toSet()); // (2)

// 使用toMap()统计学生GPA
Map<Student, Double> studentToGPA =
     students.stream().collect(Collectors.toMap(Function.identity(),// 如何生成key
                                     student -> computeGPA(student)));// 如何生成value
