# 8.21.1 回文数

Leetcode.9

## 1、题目

给你一个整数 `x` ，如果 `x` 是一个回文整数，返回 `true` ；否则，返回 `false` 。

回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

- 例如，`121` 是回文，而 `123` 不是。

**示例 1：**

```
输入：x = 121
输出：true
```



## 2、分析

解决方法有很多，这里用数学。

```
例如：1221
1221 ->cur=1
122  ->cur=12
12   ->cur=122
1    ->cur=1221
所以可以发现一个公式：cur=cur*10+num%10;num=num/10
最后判断cur==x
```

## 3、代码

```java
class Solution {
    public boolean isPalindrome(int x) {
        if(x < 0)         // 负数一定不是回文
            return false;
        
        int cur = 0;      // 用来存【反转后的数字】
        int num = x;      // 用 num 来操作，不修改原来的 x
        
        while(num != 0) {      // 把数字整个反转
            cur = cur * 10 + num % 10;  // 拼接最后一位到反转数
            num /= 10;                 // 原数去掉最后一位
        }
        
        return cur == x;   // 反转后的数字 == 原来的数字 → 回文
    }
}
```
