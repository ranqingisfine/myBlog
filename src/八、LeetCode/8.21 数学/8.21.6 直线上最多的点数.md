# 8.21.6 直线上最多的点数

leetcode.149

## 1、题目

给你一个数组 `points` ，其中 `points[i] = [xi, yi]` 表示 **X-Y** 平面上的一个点。求最多有多少个点在同一条直线上。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2021/02/25/plane2.jpg)

```
输入：points = [[1,1],[3,2],[5,3],[4,1],[2,3],[1,4]]
输出：4
```



## 2、分析

以每个点为**中心**，看其他点和它构成的**斜率**

斜率相同 → 在同一条直线上

用**分数约分 + 字符串**存斜率，避免浮点数精度错误

统计同斜率最多点，就是答案

## 3、代码

很经典，需要背下

```java
import java.util.HashMap;
import java.util.Map;

class Solution {
    public int maxPoints(int[][] points) {
        // 点的总数量
        int n = points.length;
        // 最少也有一个点，所以初始答案为1
        int ans = 1;

        // 枚举每个点作为基准点
        for (int i = 0; i < n; i++) {
            // key：斜率的最简分数形式（用字符串存储避免精度丢失）
            // value：该斜率上的点的数量
            Map<String, Integer> map = new HashMap<>();
            // 记录当前点 i 出发，同一条直线上最多的点数（不含自己）
            int max = 0;

            // 枚举其他所有点
            for (int j = i + 1; j < n; j++) {
                // 取出两个点的坐标
                int x1 = points[i][0], y1 = points[i][1];
                int x2 = points[j][0], y2 = points[j][1];

                // 计算斜率 dy/dx = (y1-y2)/(x1-x2)，直接用a、b表示分子分母
                int a = x1 - x2;
                int b = y1 - y2;

                // 求最大公约数，用于将斜率化为最简分数，避免重复
                int k = gcd(a, b);

                // 构造唯一key：最简分子_最简分母
                String key = (a / k) + "_" + (b / k);

                // 该斜率点数+1
                map.put(key, map.getOrDefault(key, 0) + 1);
                // 更新当前点出发的最大点数
                max = Math.max(max, map.get(key));
            }

            // max是同斜率点数，+1是加上基准点i本身，更新全局答案
            ans = Math.max(ans, max + 1);
        }

        return ans;
    }

    // 辗转相除法求最大公约数
    int gcd(int a, int b) {
        return b == 0 ? a : gcd(b, a % b);
    }
}
```
