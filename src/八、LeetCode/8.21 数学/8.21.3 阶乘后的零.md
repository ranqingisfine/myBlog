# 8.21.3 阶乘后的零

leetcode.172

## 1、题目

给定一个整数 `n` ，返回 `n!` 结果中尾随零的数量。

提示 `n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1`

**示例 1：**

```
输入：n = 3
输出：0
解释：3! = 6 ，不含尾随 0
```

**示例 2：**

```
输入：n = 5
输出：1
解释：5! = 120 ，有一个尾随 0
```

## 2、分析

末尾有几个 0，只需要统计：**1～n 中因数 5 的总个数**

例如：

```
n = 1010/5=2 → 答案 2
n = 2525/5=525/25=1合计 5+1=6
```



## 3、代码

```java
class Solution {
    public int trailingZeroes(int n) {
        int count = 0;
        while (n > 0) {
            n /= 5;
            count += n;
        }
        return count;
    }
}
```
