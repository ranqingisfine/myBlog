# 8.21.2 加1

Leetcode.66

## 1、题目

给定一个表示 **大整数** 的整数数组 `digits`，其中 `digits[i]` 是整数的第 `i` 位数字。这些数字按从左到右，从最高位到最低位排列。这个大整数不包含任何前导 `0`。

将大整数加 1，并返回结果的数字数组。

 **示例 1：**

```
输入：digits = [1,2,3]
输出：[1,2,4]
解释：输入数组表示数字 123。
加 1 后得到 123 + 1 = 124。
因此，结果应该是 [1,2,4]。
```

## 2、分析

从数组**最后一位**往前遍历

当前位 **+1**

- 不等于 10 → 直接返回数组
- 等于 10 → 置为 0，继续往前

如果**全部都是 9**（比如 999），最后新建一个长度 +1 的数组，第一位是 1



## 3、代码

```java
class Solution {
    public int[] plusOne(int[] digits) {
        int n = digits.length;
        // 从后往前
        for (int i = n - 1; i >= 0; i--) {
            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }
            // 等于9，加1变10，置0
            digits[i] = 0;
        }

        // 所有位都是9，新建数组
        int[] res = new int[n + 1];
        res[0] = 1;
        return res;
    }
}
```
