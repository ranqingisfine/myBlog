# 8.10.3 二叉树的层序遍历

leetcode.102

**题目**：给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。



**分析**：套用二叉树层次遍历模版即可，参考之前做过的题[8.9.7 填充每个节点的下一个右侧节点指针II](https://ranqingisfine.github.io/myBlog/%E5%85%AB%E3%80%81LeetCode/8.9%20%E4%BA%8C%E5%8F%89%E6%A0%91/8.9.7%20%E5%A1%AB%E5%85%85%E6%AF%8F%E4%B8%AA%E8%8A%82%E7%82%B9%E7%9A%84%E4%B8%8B%E4%B8%80%E4%B8%AA%E5%8F%B3%E4%BE%A7%E8%8A%82%E7%82%B9%E6%8C%87%E9%92%88II.html)



**代码**：

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> list=new ArrayList<>();
        if (root == null){
            return list;
        }
        Deque<TreeNode> q=new ArrayDeque<>();
        q.offer(root);
        while (!q.isEmpty()) {
            int size = q.size();
            List<Integer> levelList=new ArrayList<>();
            for (int i = 0; i < size; i++) {
                TreeNode curr = q.poll();
                levelList.add(curr.val);
                if (i == size-1){
                    list.add(levelList);
                }
                
                if (curr.left != null){
                    q.offer(curr.left);
                }
                if (curr.right != null){
                    q.offer(curr.right);
                }
            }
        }
        return list;
    }
}
```
