# 8.11.1 二叉搜索树的最小绝对差

leetcode.530

**题目**：

给你一个二叉搜索树的根节点 `root` ，返回 **树中任意两不同节点值之间的最小差值** 。

差值是一个正数，其数值等于两值之差的绝对值。

```
二叉搜索树：左子树<根节点<右子树
```

题目的意思是一颗二叉搜索树，求任意两个节点之间的最小绝对差



**分析**：

二叉搜索树BST+中序遍历=递增有序数组

```
        4
       / \
      2   6
     / \
    1   3
中序遍历：1 2 3 4 6
根据中序遍历顺序：
- root=1时，1的左子树为空，拿到上一个pre值计算最小值然后pre=1，1的右子树为空
- root=2时，2拿到上一个pre=1值计算最小值然后pre=2
- root=3时，3的左子树为空，拿到上一个pre=2值计算最小值然后pre=3
- root=4时，4拿到上一个pre=3值计算最小值然后pre=4
- root=6时，6的左子树为空，拿到上一个pre=4值计算最小值然后pre=6
```

最小差怎么求？最小差一定出现在相邻两个数之间！

- 中序遍历，得到升序序列
- 遍历相邻两个数，计算差值
- 记录最小的那个



**代码**：

```java
class Solution {
    int minDiff = Integer.MAX_VALUE;
    TreeNode prev = null; // 记录上一个节点

    public int getMinimumDifference(TreeNode root) {
        inorder(root);
        return minDiff;
    }

    // 中序遍历：左 → 根 → 右
    void inorder(TreeNode root) {
        if (root == null) return;

        // 1. 左
        inorder(root.left);

        // 2. 根（核心计算）
        if (prev != null) {
            minDiff = Math.min(minDiff, root.val - prev.val);
        }
        prev = root; // 更新上一个节点

        // 3. 右
        inorder(root.right);
    }
}
```
