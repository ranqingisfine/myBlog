# 8.23.6 编辑距离

leetcode.72

## 1、题目

给你两个单词 `word1` 和 `word2`， *请返回将 `word1` 转换成 `word2` 所使用的最少操作数* 。

你可以对一个单词进行如下三种操作：

- 插入一个字符
- 删除一个字符
- 替换一个字符

 **示例 1：**

```
输入：word1 = "horse", word2 = "ros"
输出：3
解释：
horse -> rorse (将 'h' 替换为 'r')
rorse -> rose (删除 'r')
rose -> ros (删除 'e')
```



## 2、分析

1、定义dp数组

```
dp[i][j]：word1前i个字符 变成 word2前j个字符的最小编辑次数
```

2、初始条件

```
- dp[0][j]：word1为空，要变成word2前j个字符->只能插入j次，dp[0][j]=j
- dp[i][0]：word2为空，word1前i个字符变空->只能删除i次，dp[i][0]=i
```

3、状态转移

```
如果 word1[i-1] == word2[j-1]，两个字符相等，不需要操作，dp[i][j]=dp[i−1][j−1]
如果 word1[i-1] != word2[j-1]，三选一，取最小值 +1（当前操作）
		替换：dp[i−1][j−1]+1
		删除(删 word1)：dp[i−1][j]+1
		插入(插 word1 = 删 word2)：dp[i][j−1]+1
dp[i][j]=min(dp[i−1][j−1],dp[i−1][j],dp[i][j−1])+1

三个操作对应关系（背这三句）
dp[i-1][j-1] +1
→ 最后一个字符替换
dp[i-1][j] +1
→ 把 word1 最后一个字符删除
dp[i][j-1] +1
→ 给 word1 插入一个字符
```



## 3、代码

```java
class Solution {
    public int minDistance(String word1, String word2) {
        int m = word1.length();
        int n = word2.length();

        // dp[i][j]：word1前i个 转为 word2前j个 的最小编辑距离
        int[][] dp = new int[m + 1][n + 1];

        // 初始化第一列：word2为空，全部删除
        for (int i = 1; i <= m; i++) {
            dp[i][0] = i;
        }

        // 初始化第一行：word1为空，全部插入
        for (int j = 1; j <= n; j++) {
            dp[0][j] = j;
        }

        // 遍历填充dp表格
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                // 当前两个字符相等
                if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                    // 无需操作，直接继承左上结果
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    // 不相等：替换、删除、插入 取最小 +1
                    int minVal = Math.min(
                        dp[i - 1][j - 1],  // 替换
                        Math.min(dp[i - 1][j], dp[i][j - 1]) // 删除 / 插入
                    );
                    dp[i][j] = minVal + 1;
                }
            }
        }

        // 全部转换完成的最小步数
        return dp[m][n];
    }
}
```
