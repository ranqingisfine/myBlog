# 8.23.4 最长回文子串

leetcode.5

## 1、题目

给你一个字符串 `s`，找到 `s` 中最长的 回文 子串。（如果字符串向前和向后读都相同，则它满足 **回文性**）

**示例 1：**

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

## 2、分析

双指针中心扩散法。

核心思想一句话：

> 回文就像**对称**，可以从**中心往两边扩散**，看最多能扩散多远。

回文有两种：

1. **奇数长度**：中心是**1 个字符**，如 `aba` → 中心 b
2. **偶数长度**：中心是**2 个字符**，如 `abba` → 中心 bb

所以我们要做的就是：

- 遍历每个位置，分别作为**奇数中心**和**偶数中心**
- 向左右扩散，直到不再对称
- 记录扩散过程中**最长**的那一段

1. 遍历字符串每个字符 `i`

2. 对每个 

   ```
   i
   ```

    做两次扩散：

   - 奇数扩散：`left = i, right = i`
   - 偶数扩散：`left = i, right = i+1`

   

3. 扩散规则：

   - 只要 `left >= 0`、`right < len`、`s[left] == s[right]`
   - 就继续 `left--, right++`

   

4. 每次扩散完，计算长度 `right - left - 1`

5. 保留**最长**的起始下标和长度

6. 最后截取字符串返回

**举例演算：s = "babad"**

字符：`b a b a d`

索引：`0 1 2 3 4`

**以 i=1（字符 a）奇数扩散**

left=1, right=1

- a == a → left=0, right=2

- b == b → left=-1, right=3

  停止。

  长度 = 3 - (-1) -1 = 3

  子串：

  ```
  bab
  ```

**以 i=2（字符 b）奇数扩散**

left=2, right=2

- b == b → left=1, right=3

- a == a → left=0, right=4

- b != d 停止

  长度 = 3

  子串：

  ```
  aba
  ```

最终最长长度 3，返回任意一个即可。



## 3、代码

```java
class Solution {
    // 记录最长回文串的起始位置和长度,为什么用 start 和 maxLen？因为字符串截取需要起始下标和长度，比存左右边界更方便。
    private int start = 0;
    private int maxLen = 0;

    public String longestPalindrome(String s) {
        int len = s.length();
        // 字符串长度小于2，本身就是最长回文
        if (len < 2) {
            return s;
        }

        // 遍历每个字符作为中心
        for (int i = 0; i < len; i++) {
            // 奇数长度回文，中心 i
            expandAroundCenter(s, i, i);
            // 偶数长度回文，中心 i 和 i+1
            expandAroundCenter(s, i, i + 1);
        }

        // 截取最长回文子串
        return s.substring(start, start + maxLen);
    }

    // 中心扩散函数
    private void expandAroundCenter(String s, int left, int right) {
        // 左右扩散，只要相等就继续
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }

        // 循环结束时，left 和 right 已经多走了一步，真实回文区间是 [left+1, right-1],最后一次满足条件的长度是 (right-1)-(left+1)+1 = right-left-1
        int curLen = right - left - 1;

        // 如果当前回文更长，更新起始位置和长度
        if (curLen > maxLen) {
            maxLen = curLen;
            start = left + 1;
        }
    }
}
```
