# 8.23.5 交错字符串

Leetcode.97

## 1、题目

给定三个字符串 `s1`、`s2`、`s3`，请你帮忙验证 `s3` 是否是由 `s1` 和 `s2` **交错** 组成的。

两个字符串 `s` 和 `t` **交错** 的定义与过程如下，其中每个字符串都会被分割成若干 **非空** 子字符串：

- `s = s1 + s2 + ... + sn`
- `t = t1 + t2 + ... + tm`
- `|n - m| <= 1`
- **交错** 是 `s1 + t1 + s2 + t2 + s3 + t3 + ...` 或者 `t1 + s1 + t2 + s2 + t3 + s3 + ...`

**注意：**`a + b` 意味着字符串 `a` 和 `b` 连接。 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/09/02/interleave.jpg)

```
输入：s1 = "aabcc", s2 = "dbbca", s3 = "aadbbcbcac"
输出：true
```



## 2、分析

题目要求：

```java
保持 s1、s2 各自内部字符的原有顺序
只能按顺序拿 s1、s2 的字符，不能打乱
```

核心思路：二维动态规划

1、DP数组定义：

```
dp[i][j]：取s1前i个字符，s2前j个字符，能否交错组成s3前i+j个字符
true：可以
false：不可以
```

 2、处理过程

```
- dp[0][0]=true，s1取0个，s2取0个，拼成空串，一定成立
- 第一行dp[0][j]：只用s2匹配s3，要求：dp[0][j]=dp[0][j - 1] && (s2.charAt(j - 1) == s3.charAt(j - 1));
- 第一列dp[i][0]：只用s1匹配s3，要求：dp[i][0]=dp[i - 1][0] && (s1.charAt(i - 1) == s3.charAt(i - 1));	
- 遍历填充整个dp数组
	// s3 当前位置下标
  int idx = i + j - 1;
  // 情况1：当前字符来自 s1
  boolean fromS1 = dp[i - 1][j] && (s1.charAt(i - 1) == s3.charAt(idx));
  // 情况2：当前字符来自 s2
  boolean fromS2 = dp[i][j - 1] && (s2.charAt(j - 1) == s3.charAt(idx));
  // 满足任意一种即可
  dp[i][j] = fromS1 || fromS2;
- 全部取完，就是最终答案：return dp[len1][len2];
```



## 3、代码

```java
public class Solution {
    public boolean isInterleave(String s1, String s2, String s3) {
        int len1 = s1.length();
        int len2 = s2.length();
        int len3 = s3.length();

        // 长度不匹配，直接不可能
        if (len1 + len2 != len3) {
            return false;
        }

        // dp[i][j]：s1前i个 + s2前j个 能否组成 s3前i+j个
        boolean[][] dp = new boolean[len1 + 1][len2 + 1];

        // 初始条件：都不取，空串匹配
        dp[0][0] = true;

        // 初始化第一列：只使用 s1
        for (int i = 1; i <= len1; i++) {
            // 当前s1字符等于s3对应字符，且上一轮状态合法
            dp[i][0] = dp[i - 1][0] && (s1.charAt(i - 1) == s3.charAt(i - 1));
        }

        // 初始化第一行：只使用 s2
        for (int j = 1; j <= len2; j++) {
            dp[0][j] = dp[0][j - 1] && (s2.charAt(j - 1) == s3.charAt(j - 1));
        }

        // 遍历填充整个dp数组
        for (int i = 1; i <= len1; i++) {
            for (int j = 1; j <= len2; j++) {
                // s3 当前位置下标
                int idx = i + j - 1;
                // 情况1：当前字符来自 s1
                boolean fromS1 = dp[i - 1][j] && (s1.charAt(i - 1) == s3.charAt(idx));
                // 情况2：当前字符来自 s2
                boolean fromS2 = dp[i][j - 1] && (s2.charAt(j - 1) == s3.charAt(idx));
                // 满足任意一种即可
                dp[i][j] = fromS1 || fromS2;
            }
        }

        // 全部取完，就是最终答案
        return dp[len1][len2];
    }
}
```
