# 8.23.2 最小路径和

leetcode.64

## 1、题目

给定一个包含非负整数的 `*m* x *n*` 网格 `grid` ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

**说明：**每次只能向下或者向右移动一步。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/11/05/minpath.jpg)

```
输入：grid = [[1,3,1],[1,5,1],[4,2,1]]
输出：7
解释：因为路径 1→3→1→1→1 的总和最小。
```



## 2、分析

规则一句话：

- 起点：左上角 `grid[0][0]`
- 终点：右下角 `grid[m-1][n-1]`
- 方向：**只能向右 或 向下**
- 目标：路径数字和**最小**

这题和 120 题三角形很像，**动态规划**是最优解，而且**原地修改网格**最简单，不用额外空间！

核心思想：

```
1.每个格子的值 = 走到这里的最小路径和
2.因为只能向右 / 向下走，所以：
3.当前格子最小和 = 当前格子值 + min (上面格子最小和，左边格子最小和)
4.最后走到右下角，它的值就是答案！
```

分情况：

网格分 3 种位置，规则不一样：

**1. 第一行（只能从左边来）**

第一行没法从上面来，**只能一直向右走**

→ 当前值 = 自己 + **左边格子的值**

**2. 第一列（只能从上面来）**

第一列没法从左边来，**只能一直向下走**

→ 当前值 = 自己 + **上面格子的值**

**3. 普通格子（从左 / 从上选小的）**

→ 当前值 = 自己 + **min (左边，上面)**

我们用这个例子讲：

```
网格：
[1,3,1]
[1,5,1]
[4,2,1]
```

起点左上角 1，终点右下角 1

**第 1 步：先填 第一行**

只能从左边来：

- 1（不变）
- 3 + 左 1 = 4
- 1 + 左 4 = 5

第一行变成：`[1,4,5]`

**第 2 步：再填 第一列**

只能从上面来：

- 1（不变）
- 1 + 上 1 = 2
- 4 + 上 2 = 6

第一列变成：`[1,2,6]`

现在网格：

```
[1,4,5]
[2,5,1]
[6,2,1]
```

**第 3 步：填普通中间格子（选左 / 上最小）**

- 5：左 = 2，上 = 4 → min=2 → 5+2=7
- 中间网格变成 `[2,7,1]`
- 2：左 = 6，上 = 7 → min=6 → 2+6=8
- 最后一列 1：左 = 8，上 = 1 → min=1 → 1+1=2

最终网格：

```
[1,4,5]
[2,7,6]
[6,8,7]
```

**终点 7 就是答案！**

最小路径：1 → 1 → 2 → 1，和为 7

## 3、代码

```java
class Solution {
    public int minPathSum(int[][] grid) {
        // 1. 获取网格的行数和列数
        int m = grid.length;    // 行数
        int n = grid[0].length; // 列数

        // 2. 填充 第一行（只能从左边来）
        for (int j = 1; j < n; j++) {
            // 当前值 = 自己 + 左边的值
            grid[0][j] += grid[0][j - 1];
        }

        // 3. 填充 第一列（只能从上面来）
        for (int i = 1; i < m; i++) {
            // 当前值 = 自己 + 上面的值
            grid[i][0] += grid[i - 1][0];
        }

        // 4. 填充 中间普通格子（从左/上选最小的）
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                // 取 左边 和 上面 的最小值，加上自己
                grid[i][j] += Math.min(grid[i][j - 1], grid[i - 1][j]);
            }
        }

        // 5. 右下角就是最小路径和
        return grid[m - 1][n - 1];
    }
}
```
