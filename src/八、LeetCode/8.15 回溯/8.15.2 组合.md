# 8.15.2 组合

Leetcode.77

## 1、题目

给定两个整数 `n` 和 `k`，返回范围 `[1, n]` 中所有可能的 `k` 个数的组合。

你可以按 **任何顺序** 返回答案。

**示例 1：**

```
输入：n = 4, k = 2
输出：
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```



## 2、分析

用回溯，每次选一个数，后面只能选比它大的数，避免重复。

```
例如：n=4，k=2
		[1,2,3,4]    for(1~4)
		/|\
	[2  3 4]       for(2~4)
```





## 3、代码

```java
class Solution {
    public List<List<Integer>> combine(int n, int k) {
        List<List<Integer>> list = new ArrayList<>();
        dfs(n, k, new ArrayList<>(), list, 1);
        return list;
    }

    private void dfs(int n, int k, List<Integer> path, List<List<Integer>> list, int start) {
        // 终止条件：够 k 个数
        if (path.size() == k) {
            list.add(new ArrayList<>(path)); // ✅ 存拷贝
            return;
        }

        for (int i = start; i <= n; i++) {
            path.add(i);
            dfs(n, k, path, list, i + 1); // ✅ 传 i+1
            path.remove(path.size() - 1);
        }
    }
}
```
