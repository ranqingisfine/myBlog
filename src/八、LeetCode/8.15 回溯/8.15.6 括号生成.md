# 8.15.6 括号生成

Leetcode.22

## 1、题目

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。

**示例 1：**

```
输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
```



## 2、分析

```
开始递归
├── 左括号没满？→ 加左括号，递归
└── 右括号 < 左括号？→ 加右括号，递归
直到左右括号都等于n → 保存结果
```

核心规则：

- 左括号：没满就加
- 右括号：必须比左括号少才能加



## 3、代码

```java
import java.util.ArrayList;
import java.util.List;

class Solution {
    // 主方法：入口，生成n对合法括号
    public List<String> generateParenthesis(int n) {
        // 用来存放最终所有合法的括号结果
        List<String> list = new ArrayList<>();
        // 开始递归（深度优先搜索DFS）
        // 参数：当前字符串为空，左括号用了0个，右括号用了0个，总对数n，结果集合list
        dfs("", 0, 0, n, list);
        // 返回最终答案
        return list;
    }

    /**
     * 递归/回溯核心方法
     * @param cur   当前已经拼接好的括号字符串
     * @param left  已经使用的左括号数量
     * @param right 已经使用的右括号数量
     * @param n     一共需要生成多少对括号
     * @param list  存放所有合法结果
     */
    private void dfs(String cur, int left, int right, int n, List<String> list){
        // ===================== 递归终止条件（出口） =====================
        // 如果左括号和右括号都刚好用满 n 个
        if (left == n && right == n){
            // 说明得到了一个合法的括号组合，加入结果集
            list.add(cur);
            // 结束当前递归，返回上一层
            return;
        }

        // ===================== 递归选择（生成左括号） =====================
        // 条件：左括号还没到n个，可以继续加左括号
        if (left < n){
            // 选择：在当前字符串后加一个"("
            // 左括号数量 +1，右括号不变
            // 这里直接传 left+1，不会修改原来的left变量，递归回来自动回溯
            dfs(cur + "(", left + 1, right, n, list);
        }

        // ===================== 递归选择（生成右括号） =====================
        // 条件：右括号数量 必须 小于 左括号数量（保证合法，不会出现 )( 这种情况）
        if (right < left){
            // 选择：在当前字符串后加一个")"
            // 右括号数量 +1，左括号不变
            dfs(cur + ")", left, right + 1, n, list);
        }
    }
}
```
