# 8.15.4 组合总和

Leetcode.39

## 1、题目

给你一个 **无重复元素** 的整数数组 `candidates` 和一个目标整数 `target` ，找出 `candidates` 中可以使数字和为目标数 `target` 的 所有 **不同组合** ，并以列表形式返回。你可以按 **任意顺序** 返回这些组合。

`candidates` 中的 **同一个** 数字可以 **无限制重复被选取** 。如果至少一个数字的被选数量不同，则两种组合是不同的。 

对于给定的输入，保证和为 `target` 的不同组合数少于 `150` 个。



## 2、分析

1. **可以重复选同一个数字**

   比如 [2,2,3] 是允许的。

   

2. **不能有顺序不同的重复组合**

   [2,3] 和 [3,2] 算同一个，只能出现一次。

   → 所以用 **start**，只往后选，不回头。

   

3. **递归怎么传**

   选了第 i 个数，下一层继续从 **i** 开始，不是 i+1

   这样就能重复选当前数，又不会回头乱选导致重复。

举例：

```
candidates = [2,3,6,7]
```

- 如果你选了 2

  下一轮只能选：2、3、6、7

- 如果你选了 3

  下一轮只能选：3、6、7

- 如果你选了 6

  下一轮只能选：6、7

**永远不能回头选前面的数字！**

这样就**绝对不会出现**

`[2,3]` 和 `[3,2]` 这种重复组合

## 3、代码

```java
class Solution {
    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        List<List<Integer>> list=new ArrayList<>();
        dfs(candidates,target,new ArrayList<>(),0,list,0);
        return list;
    }
    private void dfs(int[] candidates,int target,List<Integer> path,int sum,List<List<Integer>> list,int start){
        if (sum > target){
            return;
        }
        if (sum == target){
            list.add(new ArrayList<>(path));
            return;
        }
        //✅ 从 start 开始，不是 0
        for (int i = start; i < candidates.length; i++) {
            path.add(candidates[i]);
            sum=sum+candidates[i];
            dfs(candidates,target,path,sum,list,i);
            path.remove(path.size()-1);
            sum-=candidates[i];
        }
    }
}
```
