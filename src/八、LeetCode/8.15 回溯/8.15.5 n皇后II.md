# 8.15.5 n皇后II

Leetcode.52

## 1、题目

**n 皇后问题** 研究的是如何将 `n` 个皇后放置在 `n × n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n` ，返回 **n 皇后问题** 不同的解决方案的数量。

题目要求：

给定一个整数 `n`，代表 `n × n` 的棋盘。

将 `n` 个皇后放在棋盘上，使得：

- 任意两个皇后**不能互相攻击**

皇后攻击范围：（同1行、同1列、两条斜线不能有皇后）

- 同一行
- 同一列
- 同一斜线（两条：左上→右下、右上→左下）

题目**不需要输出具体棋盘**，只要求返回：

- 有多少种合法的摆放方案

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/11/13/queens.jpg)

```
输入：n = 4
输出：2
解释：如上图所示，4 皇后问题存在两个不同的解法。
```

**示例 2：**

```
输入：n = 1
输出：1
```

## 2、分析

整个流程就是从上到下一行一行的放皇后，例如第一行某列放好皇后后，再往第二行的合适列放皇后，所以只用管列占用、两条斜线占用

**1️⃣ 棋盘坐标**

假设 `n = 4`。

我们用 `(row, col)` 表示位置：

```
(0,0) (0,1) (0,2) (0,3)
(1,0) (1,1) (1,2) (1,3)
(2,0) (2,1) (2,2) (2,3)
(3,0) (3,1) (3,2) (3,3)
```

**2️⃣ 皇后为什么会冲突**

皇后攻击：

- 同一行
- 同一列
- 同一斜线

因为代码是一行一行递归：

```
backtrack(row + 1, ...)
```

所以：

✅ 行冲突天然不存在
 （每次只放当前这一行一个皇后）

因此只需要判断：

- 列
- 两种斜线

**3️⃣ 第一种斜线：****`\`**

代码里的：

```]
int d1 = row - c + n;
```

对应这种方向：

```
\ 方向
```

例如：

```
(0,0)
    (1,1)
        (2,2)
            (3,3)
```

观察一下：

```
0-0 = 0
1-1 = 0
2-2 = 0
3-3 = 0
```

**规律：**

**同一条** **`\`** **斜线：**

```
row - col 相同
```

再看另一条：

```
(0,1)
    (1,2)
        (2,3)
```

```
0-1 = -1
1-2 = -1
2-3 = -1
```

还是一样。

所以：

```
d1 = row - col
```

就能唯一表示一条 `\` 斜线。

**4️⃣ 为什么要** **`+ n`**

因为：

```
row - col
```

可能是负数。

例如：

```
(0,3)
0 - 3 = -3
```

数组下标不能负数。

所以：

```
d1 = row - col + n;
```

整体右移。

**5️⃣ 第二种斜线：****`/`**

代码：

```
int d2 = row + c;
```

对应：

```
/ 方向
```

例如：

```
        (0,3)
    (1,2)
(2,1)
```

观察：

```
0+3 = 3
1+2 = 3
2+1 = 3
```

再看：

```
(0,2)
    (1,1)
        (2,0)
```

```
0+2 = 2
1+1 = 2
2+0 = 2
```

**规律：**

**同一条** **`/`** **斜线：**

```
row + col 相同
```

所以：

```
d2 = row + col;
```

**6️⃣ boolean数组到底表示什么**

表示某条 \ 斜线 是否已经有皇后

例如：

```
diag1[4] = true;
表示row - col + n == 4
这整条 \ 斜线已经被占用了。
后面谁再落到这条线上：
if (diag1[d1]) continue;
直接跳过
```

## 3、代码

```java
class Solution {
    int count = 0;//全局变量
    public int totalNQueens(int n) {
        boolean[] col = new boolean[n];         // 列是否占用
        boolean[] diag1 = new boolean[2 * n];    // \ 斜线数组
        boolean[] diag2 = new boolean[2 * n];    // / 斜线数组
        //行数是通过递归控制
        backtrack(0, n, col, diag1, diag2);
        return count;
    }

    private void backtrack(int row, int n, boolean[] col, boolean[] diag1, boolean[] diag2) {
        // 放完所有行，找到一种解法
        if (row == n) {
            count++;
            return;
        }
        //遍历当前行的所有列
        for (int c = 0; c < n; c++) {
            int d1 = row - c + n; // 加n防止负数
            int d2 = row + c;
            if (col[c] || diag1[d1] || diag2[d2]) continue;

            // 放皇后
            col[c] = true;
            diag1[d1] = true;
            diag2[d2] = true;

            backtrack(row + 1, n, col, diag1, diag2);

            // 回溯
            col[c] = false;
            diag1[d1] = false;
            diag2[d2] = false;
        }
    }
}
```
