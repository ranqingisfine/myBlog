# 8.15.5 n皇后II

Leetcode.52

## 1、题目

**n 皇后问题** 研究的是如何将 `n` 个皇后放置在 `n × n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n` ，返回 **n 皇后问题** 不同的解决方案的数量。

题目要求：

给定一个整数 `n`，代表 `n × n` 的棋盘。

将 `n` 个皇后放在棋盘上，使得：

- 任意两个皇后**不能互相攻击**

皇后攻击范围：（同1行、同1列、两条斜线不能有皇后）

- 同一行
- 同一列
- 同一斜线（两条：左上→右下、右上→左下）

题目**不需要输出具体棋盘**，只要求返回：

- 有多少种合法的摆放方案



## 2、分析

- 对角线 1（`\` 型）：`row - col` 相同，处理的时候后面+n，防止负数
- 对角线 2（`/` 型）：`row + col` 相同

用两个 boolean 数组标记是否被占。

理解就是可以用数组的一个值来表示一条对角线

```
完整思路：
从第 0 行开始递归
尝试在当前行的每一列放皇后
如果该列、两条斜线都没被占，就放
标记列 + 两条斜线为已使用
递归下一行
回溯：取消标记
走到第 n 行，说明找到一种解法，计数 +1
```

整个流程就是从上到下一行一行的放皇后，例如第一行某列放好皇后后，再往第二行的合适列放皇后，所以只用管列占用、两条斜线占用

## 3、代码

```java
class Solution {
    int count = 0;//全局变量
    public int totalNQueens(int n) {
        boolean[] col = new boolean[n];         // 列是否占用
        boolean[] diag1 = new boolean[2 * n];    // \ 斜线
        boolean[] diag2 = new boolean[2 * n];    // / 斜线
        //行数是通过递归控制
        backtrack(0, n, col, diag1, diag2);
        return count;
    }

    private void backtrack(int row, int n, boolean[] col, boolean[] diag1, boolean[] diag2) {
        // 放完所有行，找到一种解法
        if (row == n) {
            count++;
            return;
        }
        //遍历当前行的所有列
        for (int c = 0; c < n; c++) {
            int d1 = row - c + n; // 加n防止负数
            int d2 = row + c;
            if (col[c] || diag1[d1] || diag2[d2]) continue;

            // 放皇后
            col[c] = true;
            diag1[d1] = true;
            diag2[d2] = true;

            backtrack(row + 1, n, col, diag1, diag2);

            // 回溯
            col[c] = false;
            diag1[d1] = false;
            diag2[d2] = false;
        }
    }
}
```
