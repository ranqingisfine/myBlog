# 8.9.7 二叉树展开为链表

leetcode.114

**题目**：给你二叉树的根结点 `root` ，请你将它展开为一个单链表：

- 展开后的单链表应该同样使用 `TreeNode` ，其中 `right` 子指针指向链表中下一个结点，而左子指针始终为 `null` 。
- 展开后的单链表应该与二叉树 [**先序遍历**](https://baike.baidu.com/item/先序遍历/6442839?fr=aladdin) 顺序相同。



**分析**：

```
    1
   / \
  2   5
 / \   \
3   4   6

```

**第一步：处理节点 1**

- 有左子树 2
- 找到左子树最右下角：**4**
- 把 1 的右子树（5→6）接到 4 后面
- 把左子树 2 搬到右边
- 左边置空

现在变成：

```
1
 \
  2
 / \
3   4
     \
      5
       \
        6
```

**第二步：处理节点 2**

- 有左子树 3
- 找到最右下角 3
- 把 2 的右子树（4→…）接到 3 后面
- 左子树搬到右边
- 左边置空

现在变成：

```
1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6
```

一句话总结：把左子树挂到右边 → 把原来的右子树接到左子树的最后一个节点下面 

因为先序遍历是根节点、左子树、右子树，所以左子树的最右下节点应该链接右子树，不断的的操作左子树直到左子树都为空。



**代码**：

```java
class Solution {
    public void flatten(TreeNode root) {
        // 遍历每个节点
        while (root != null) {
            // 1. 如果没有左子树，直接往右走
            if (root.left == null) {
                root = root.right;
            } 
            // 2. 有左子树，就要处理
            else {
                // 找到左子树最右下角的节点
                TreeNode pre = root.left;
                while (pre.right != null) {
                    pre = pre.right;
                }
                
                // 把 root 的右子树 接到 左子树最右下角
                pre.right = root.right;
                
                // 把左子树搬到右边
                root.right = root.left;
                
                // 左子树置空
                root.left = null;
                
                // 继续处理下一个节点
                root = root.right;
            }
        }
    }
}
```
