# 8.9.7 填充每个节点的下一个右侧节点指针II

leetCode.117

## 1、层序遍历

什么是层序遍历？一层一层地遍历二叉树，从上到下，从左到右。

如何实现？靠队列，出队一个，就把它的左右孩子入队

```
        1         → 第一层
      /   \
     2     3      → 第二层
    / \     \
   4  5      6    → 第三层
我们以上面的树为例：
初始状态
队列：[1]
第一步：出队 1
把 1 拿出来把 1 的左、右 2、3 入队队列变成：[2, 3]
第二步：出队 2
把 2 拿出来把 2 的左、右 4、5 入队队列变成：[3, 4, 5]
第三步：出队 3
把 3 拿出来把 3 的右 6 入队队列变成：[4, 5, 6]
第四步：出队 4 → 5 → 6
没有孩子了，队列变空
```

标准代码（背下来）：

```java
public void levelOrder(TreeNode root) {
    if (root == null) return;
    
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);  // 根节点入队
    
    while (!q.isEmpty()) {
        TreeNode cur = q.poll(); // 出队
        System.out.println(cur.val); // 访问
        
        // 左孩子入队
        if (cur.left != null) {
            q.offer(cur.left);
        }
        // 右孩子入队
        if (cur.right != null) {
            q.offer(cur.right);
        }
    }
}
```



## 2、填充每个节点的下一个右侧节点指针II

```
        1
      /   \
     2     3
    / \     \
   4   5     6
队列一开始：[1]size = 1 → for 循环 1 次→ 处理第 1 层
处理完 1，队列变成：[2,3]
下一轮循环：size = 2 → for 循环 2 次→ 处理第 2 层（2 和 3）
处理完，队列：[4,5,6]
下一轮循环：size = 3 → for 循环 3 次→ 处理第 3 层（4、5、6）
```



**代码**：

```java
class Node {
    public int val;
    public Node left;
    public Node right;
    public Node next;

    public Node() {}
    public Node(int _val) { val = _val; }
    public Node(int _val, Node _left, Node _right, Node _next) {
        val = _val;
        left = _left;
        right = _right;
        next = _next;
    }
};

class Solution {
    public Node connect(Node root) {
        if (root == null) return null;

        // 1. 队列：存每一层的节点
        Queue<Node> q = new LinkedList<>();
        q.offer(root);

        // --------------------------
        // 轮询一层一层处理
        // --------------------------
        while (!q.isEmpty()) {
            // 🔥 关键：记住当前层有多少个节点
            int levelSize = q.size();

            // 🔥 关键：只处理当前这一层！！！
            for (int i = 0; i < levelSize; i++) {
                Node cur = q.poll();

                // --------------------------
                // 核心：连接 next 指针
                // --------------------------
                if (i != levelSize - 1) {
                    // 不是最后一个 → 指向队列里下一个节点
                    cur.next = q.peek();
                } else {
                    // 最后一个 → 指向 null
                    cur.next = null;
                }

                // 把下一层放进队列
                if (cur.left != null) {
                    q.offer(cur.left);
                }
                if (cur.right != null) {
                    q.offer(cur.right);
                }
            }
        }

        return root;
    }
}
```
