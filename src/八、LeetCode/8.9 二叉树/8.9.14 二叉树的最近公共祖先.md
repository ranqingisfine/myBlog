# 8.9.14 二叉树的最近公共祖先

leetcode.236

**题目**：给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

[百度百科](https://baike.baidu.com/item/最近公共祖先/8918834?fr=aladdin)中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（**一个节点也可以是它自己的祖先**）。”

题目理解：

```
给你一棵二叉树，和两个节点 p、q找出离他俩最近的、共同的祖先节点。
规则：
一个节点可以算是自己的祖先
保证 p、q 一定在树里
```



**分析**：

后序遍历：先问左，再问右，最后判断自己

给每个节点定义一个功能：如果我这棵树里能找到p或q，我就返回自己，否则返回null

三种情况：

- 左边能找到，右边能找到，我就是最近公共祖先
- 只有左边找到，答案在左边
- 只有右边找到，答案在右边

情况1：

```
        3
      /   \
     5     1
    / \   / \
   6  2  0  8
     / \
    7  4
假设要找p=5,q=1
- 递归查左子树节点5，发现就是节点p，直接返回5上去（停止左子树遍历）
- 递归查右子树节点1，发现就是节点q，直接返回1上去（停止右子树遍历）
- 回到根节点3做判断，左边返回了5，右边返回了1，两边都有值，则返回根节点
```

情况2:

```
        3
      /   \
     5     1
    / \   / \
   6   2 0   8
      / \
     7   4
 假设要找p=5,q=4
 - 递归查左子树节点5，发现就是节点p，直接返回5上去（停止左子树遍历）
 - 递归查右子树节点1，整颗右子树递归完返回null上去
 - 回到根节点3做判断，左边返回了5，右边返回了null，则返回右值的节点5（为什么是返回5，也就是它本身，是因为这p和q本身就是二叉树里的节点，如果右子树都没有的话，左子树找到了其中一个p，那么另一个节点肯定是p的子节点）
```



**代码**：

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        // 1. 空节点返回 null
        if (root == null) {
            return null;
        }

        // 2. 如果当前节点就是 p 或 q，直接返回自己
        if (root == p || root == q) {
            return root;
        }

        // ==========================
        // 后序遍历开始！
        // ==========================

        // 3. 先查左子树
        TreeNode left = lowestCommonAncestor(root.left, p, q);
        
        // 4. 再查右子树
        TreeNode right = lowestCommonAncestor(root.right, p, q);
        
        // 5. 最后判断自己（根）
        if (left != null && right != null) {
            // 左右都找到了 → 我就是答案！
            return root;
        }
        else if (left != null) {
            // 只有左边有 → 答案在左边
            return left;
        }
        else {
            // 只有右边有 → 答案在右边
            return right;
        }
    }
}
```
