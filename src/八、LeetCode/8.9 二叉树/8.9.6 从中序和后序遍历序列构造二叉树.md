# 8.9.6 从中序和后序遍历序列构造二叉树

leetcode.106

解题思路和[8.9.5 从前序遍历与中序遍历序列构造二叉树](https://ranqingisfine.github.io/myBlog/%E5%85%AB%E3%80%81LeetCode/8.9%20%E4%BA%8C%E5%8F%89%E6%A0%91/8.9.5%20%E4%BB%8E%E5%89%8D%E5%BA%8F%E9%81%8D%E5%8E%86%E4%B8%8E%E4%B8%AD%E5%BA%8F%E9%81%8D%E5%8E%86%E5%BA%8F%E5%88%97%E6%9E%84%E9%80%A0%E4%BA%8C%E5%8F%89%E6%A0%91.html)一样

**代码**：

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    private Map<Integer,Integer> inMap=new HashMap<>();
    private int[] postorder;
    private int postIndex;
    public TreeNode buildTree(int[] inorder,int[] postorder){
        this.postorder=postorder;
        postIndex=postorder.length-1;
        for (int i=0;i<inorder.length;i++){
            inMap.put(inorder[i],i);
        }
        
        return build(0,inorder.length-1);
    }
    private TreeNode build(int left,int right){
        if (left>right){
            return null;
        }
        int rootVal = postorder[postIndex];
        postIndex--;
        TreeNode root=new TreeNode(rootVal);
        Integer mid = inMap.get(rootVal);
      //注意先访问right
        root.right=build(mid+1,right);
        root.left=build(left,mid-1);
        return root;
    }
}
```
