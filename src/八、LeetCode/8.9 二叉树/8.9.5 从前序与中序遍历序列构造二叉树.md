# 8.9.5 从前序与中序遍历序列构造二叉树

leetCode.105

## 1、前序遍历和中序遍历

```
        1
      /   \
     2     3
    / \
   4   5
   
前序遍历（根左右）：1 2 4 5 3
中序遍历（左根右）：4 2 5 1 3
```

关键的递归逻辑

```shell
# 前序遍历
print(root);
preOrder(left);
preOrder(right);
## 中序遍历
inOrder(left);
print(root);
inOrder(right);
```

根据前序遍历和中序遍历构造二叉树

```
前序遍历（根左右）：1 2 4 5 3
中序遍历（左根右）：4 2 5 1 3

        1

     2     3

   4   5	
```



## 2、从前序遍历与中序遍历序列构造二叉树

**题目**：

给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的**先序遍历**， `inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点。



**分析**：

```
前序遍历（根左右）：1 2 4 5 3
中序遍历（左根右）：4 2 5 1 3
----
1.第1次，1是根节点时，找到中序遍历值为1的下标，将数组分为两部分：4 2 5和3
2.处理4 2 5，此时2为根节点，将数组分为两部分：4和5
3.处理4，此时4为根节点，4的左子树和右子树都为空，回溯到根节点2，此时2的左子树指向4
4.处理5，此时5为根节点，5的左子树和右子树都为空，回溯到根节点2，此时2的右子树指向4
5.回溯到根节点1，此时1的左子树指向2
6.处理3，此时3为根节点，3的左子树和右子树都为空，回溯到根节点1，此时1的右子树指向3
递归完毕
```



**代码**：

```java
import java.util.HashMap;
import java.util.Map;

class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

class Solution {
    // 全局缓存：方便快速找到中序里根节点的位置
    private Map<Integer, Integer> inOrderMap;
    private int preIndex; // 前序遍历的指针（一直往后取根）
    private int[] preorder;

    public TreeNode buildTree(int[] preorder, int[] inorder) {
        this.preorder = preorder;
        this.preIndex = 0; // 最开始指向根（前序第一个）
        this.inOrderMap = new HashMap<>();

        // 把中序存入map：key=值，value=下标
        for (int i = 0; i < inorder.length; i++) {
            inOrderMap.put(inorder[i], i);
        }

        // 递归构建：中序的范围是 0 ~ len-1
        return build(0, inorder.length - 1);
    }

    // left, right：当前在中序里的区间范围
    private TreeNode build(int left, int right) {
        // 递归终止：区间不合法，没有节点
        if (left > right) {
            return null;
        }

        // 1. 前序取根
        int rootVal = preorder[preIndex];
        preIndex++; // 指针后移
        TreeNode root = new TreeNode(rootVal);

        // 2. 找到根在中序里的位置
        int mid = inOrderMap.get(rootVal);

        // 3. 递归构建左子树（中序左边）
        root.left = build(left, mid - 1);

        // 4. 递归构建右子树（中序右边）
        root.right = build(mid + 1, right);

        return root;
    }
}
```
