# 8.9.10 求根节点到叶子节点数字之和

leetcode.129

**题目**：给你一个二叉树的根节点 `root` ，树中每个节点都存放有一个 `0` 到 `9` 之间的数字。

每条从根节点到叶节点的路径都代表一个数字：

- 例如，从根节点到叶节点的路径 `1 -> 2 -> 3` 表示数字 `123` 。

计算从根节点到叶节点生成的 **所有数字之和** 。

**叶节点** 是指没有子节点的节点。



**分析**：

用先序遍历的模版



**代码**：

```java
public int sumNumbers(TreeNode root) {
    return cal(root, 0); // ✅ 这里必须传 0
}

private int cal(TreeNode root, int val) {
    if (root == null) {
        return 0;
    }
    
    // 当前层先把数字算好
    int currSum = val * 10 + root.val;
    
    // 到叶子节点，直接返回这条路径的数字
    if (root.left == null && root.right == null) {
        return currSum;
    }
    
    // 递归左右
    int left = cal(root.left, currSum);
    int right = cal(root.right, currSum);
    
    return left + right;
}
```
