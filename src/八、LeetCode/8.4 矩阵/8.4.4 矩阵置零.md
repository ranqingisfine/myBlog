# 8.4.4 矩阵置零

Leetcode.73



**题目**：给定一个 `*m* x *n*` 的矩阵，如果一个元素为 **0** ，则将其所在行和列的所有元素都设为 **0** 。请使用 **[原地](http://baike.baidu.com/item/原地算法)** 算法**。**



**分析**：**先记录，后置零”**：

1. 第一步：遍历矩阵，用两个哈希集合分别记录 “包含 0 的行号” 和 “包含 0 的列号”；
2. 第二步：遍历记录的行号，将这些行的所有元素置为 0；
3. 第三步：遍历记录的列号，将这些列的所有元素置为 0；

**代码**：

```java
class Solution {
    public void setZeroes(int[][] matrix) {
int row=matrix.length;
        int col=matrix[0].length;
        //为0的行号
        HashSet<Integer> rowSet = new HashSet<>();
        HashSet<Integer> colSet = new HashSet<>();
        //遍历每一个元素
        for (int i = 0; i < row; i++) {
            for (int j = 0; j < col; j++) {
                if (matrix[i][j]==0){
                    rowSet.add(i);
                    colSet.add(j);
                }
            }
        }
        //将所有行号置0
        for (Integer val : rowSet) {
            for (int i = 0; i < col; i++) {
                matrix[val][i]=0;
            }
        }
        //所有列号置0
        for (Integer val : colSet) {
            for (int i = 0; i < row; i++) {
                matrix[i][val]=0;
            }
        }
    }
}
```
