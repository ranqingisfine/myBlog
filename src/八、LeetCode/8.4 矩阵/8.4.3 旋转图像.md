# 8.4.3 旋转图像

Leetcode.48

**题目**：

给定一个 *n* × *n* 的二维矩阵 `matrix` 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在**[ 原地](https://baike.baidu.com/item/原地算法)** 旋转图像，这意味着你需要直接修改输入的二维矩阵。**请不要** 使用另一个矩阵来旋转图像。



**分析**：

原图像--->水平翻转--->左上右下对角交换--->90翻转后的图像



**代码**：

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n=matrix.length;
        //水平交换
        for (int i = 0; i < n / 2; i++) {
            for (int j = 0; j < n; j++) {
                int temp=matrix[i][j];
                matrix[i][j]=matrix[n-i-1][j];
                matrix[n-i-1][j]=temp;
            }
        }
        //对角交换
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < i; j++) {
                int temp=matrix[i][j];
                matrix[i][j]=matrix[j][i];
                matrix[j][i]=temp;
            }
        }
    }
}
```
