# 8.18.7 寻找两个正序数组的中位数

leetcode.3

## 1、题目

给定两个大小分别为 `m` 和 `n` 的正序（从小到大）数组 `nums1` 和 `nums2`。请你找出并返回这两个正序数组的 **中位数** 。

算法的时间复杂度应该为 `O(log (m+n))` 。

**示例 1：**

```
输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2
```

**示例 2：**

```
输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5
```

简单说就是将两个数组合并，如果个数是奇数，则取中间那个数；如果个数是偶数则取中间两个数的平均值



## 2、分析

采用最简单的方式，新建一个merge数组将两个数组按照递增顺序合并，然后判断，如果merge总个数是奇数，则取中间那个数；如果是偶数则取中间两个数的平均值

## 3、代码

```java
class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        int len1=nums1.length;
        int len2=nums2.length;
        int i=0;
        int total=len1+len2;
        int[] merge=new int[total];
        int m=0;
        int n=0;
        while(m<len1 && n<len2){
            if (nums1[m]<nums2[n]){
                merge[i]=nums1[m];
                m++;
            }else{
                merge[i]=nums2[n];
                n++;
            }
            i++;
        }
        if (m==len1){
            //要将len2的剩余元素添加进去
            while(n<len2){
                merge[i]=nums2[n];
                i++;
                n++;
            }
        }
        if (n==len2){
            while (m<len1){
                merge[i]=nums1[m];
                i++;
                m++;
            }
        }
        if (total%2 == 0){
            //偶数
            return (merge[(total-1)/2]+merge[((total-1)/2)+1])/2.0;
        }else{
            //奇数
            return merge[(total-1)/2];
        }
    }
}
```
