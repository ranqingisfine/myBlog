# 8.18.2 搜索二维矩阵

Leetcode.74

## 1、题目

给你一个满足下述两条属性的 `m x n` 整数矩阵：

- 每行中的整数从左到右按非严格递增顺序排列。
- 每行的第一个整数大于前一行的最后一个整数。

给你一个整数 `target` ，如果 `target` 在矩阵中，返回 `true` ；否则，返回 `false` 。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/10/05/mat.jpg)

```
输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
输出：true
```



## 2、分析

- 取每行0列组成一个集合，对这个集合进行二分，确定target归属于哪一行
- 对已确定的那行再次进行二分，判断是否有target

## 3、代码

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int rows = matrix.length;
        List<Integer> rowList=new ArrayList<>();
        for (int i = 0; i < rows; i++) {
            rowList.add(matrix[i][0]);
        }
        //行，二分
        int left=0;
        int right=rowList.size()-1;
        while (left<=right){
            int mid=left+(right-left)/2;
            if (rowList.get(mid) == target){
                return true;
            }else if (rowList.get(mid) < target){
                left=mid+1;
            }else{
                right=mid-1;
            }
        }
                // 防止极端情况：target 比第一行第一个还小，直接返回 false
        if (right < 0) return false;
        //值在right行里面，对left行的数据进行二分查找
        int fRight = matrix[right].length-1;
        int fLeft=0;
        while (fLeft<=fRight){
            int mid=fLeft+(fRight-fLeft)/2;
            if (matrix[right][mid] == target){
                return true;
            }else if (matrix[right][mid] < target){
                fLeft=mid+1;
            }else{
                fRight=mid-1;
            }
        }
        return false;
    }
}
```
