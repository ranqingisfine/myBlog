# 8.13.3 单词接龙

Leetcode.133

## 1、题目

字典 `wordList` 中从单词 `beginWord` 到 `endWord` 的 **转换序列** 是一个按下述规格形成的序列 `beginWord -> s1 -> s2 -> ... -> sk`：

- 每一对相邻的单词只差一个字母。
-  对于 `1 <= i <= k` 时，每个 `si` 都在 `wordList` 中。注意， `beginWord` 不需要在 `wordList` 中。
- `sk == endWord`

给你两个单词 `beginWord` 和 `endWord` 和一个字典 `wordList` ，返回 *从 `beginWord` 到 `endWord` 的 **最短转换序列** 中的 **单词数目*** 。如果不存在这样的转换序列，返回 `0` 。



题目分析：

```
1.beginWord → endWord，每次改一个字母
2.中间必须是 wordList 里的单词
3.求最短转换序列长度（起点算 1）
4.不存在返回 0
```



## 2、分析

此题和[8.13.2 最小基因变化](https://ranqingisfine.github.io/myBlog/%E5%85%AB%E3%80%81LeetCode/8.13%20%E5%9B%BE%E7%9A%84%E5%B9%BF%E5%BA%A6%E4%BC%98%E5%85%88%E6%90%9C%E7%B4%A2/8.13.2%20%E6%9C%80%E5%B0%8F%E5%9F%BA%E5%9B%A0%E5%8F%98%E5%8C%96.html)一模一样



## 3、代码

```java
class Solution {
    public int ladderLength(String beginWord, String endWord, List<String> wordList) {
        Set<String> set = new HashSet<>(wordList);
        if (!set.contains(endWord)) return 0;

        Queue<String> q = new LinkedList<>();
        Set<String> visited = new HashSet<>();

        q.offer(beginWord);
        visited.add(beginWord);
        int level = 1;  // 起点算长度1

        while (!q.isEmpty()) {
            int size = q.size();
            for (int i = 0; i < size; i++) {
                String cur = q.poll();

                // 到达终点，返回当前长度
                if (cur.equals(endWord)) return level;

                char[] arr = cur.toCharArray();
                // 遍历每一位
                for (int j = 0; j < arr.length; j++) {
                    char old = arr[j];
                    // 换成 a-z 所有字母
                    for (char c = 'a'; c <= 'z'; c++) {
                        if (c == old) continue;
                        arr[j] = c;
                        String next = new String(arr);
                        // 合法且没访问过
                        if (set.contains(next) && !visited.contains(next)) {
                            visited.add(next);
                            q.offer(next);
                        }
                    }
                    arr[j] = old; // 还原
                }
            }
            level++;
        }
        return 0; // 没找到
    }
}
```
