# 8.1.20 最长公共前缀

**题目**：

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

**分析**：

先总结这段代码的整体功能：以数组第一个字符串为初始前缀，逐个与后续字符串纵向对比字符，逐步缩短公共前缀，直到前缀为空或遍历完所有字符串。

**代码**：

```java
class Solution {
    public String longestCommonPrefix(String[] strs) {
        //初始前缀
        String flag=strs[0];
        for (int i = 1; i < strs.length; i++) {
            String str=strs[i];
          	//前缀的最后一个下标
            int j=0;
            //遍历两个字符串的公共长度
            for (; j < str.length() && j<flag.length(); j++) {
                if (flag.charAt(j) != str.charAt(j)){
                    break;
                }
            }
            flag=flag.substring(0,j);
          	//如果任意一个字符串都没有匹配上，就直接返回结果
            if (flag.equals("")){
                return flag;
            }
        }
        return flag;
    }
}
```
