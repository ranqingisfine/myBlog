# 8.1.15 分发糖果

leetCode.135

**题目**：`n` 个孩子站成一排。给你一个整数数组 `ratings` 表示每个孩子的评分。

你需要按照以下要求，给这些孩子分发糖果：

- 每个孩子至少分配到 `1` 个糖果。
- 相邻两个孩子中，评分更高的那个会获得更多的糖果。

请你给每个孩子分发糖果，计算并返回需要准备的 **最少糖果数目** 。



**分析**：

规则定义：设学生 *A* 和学生 *B* 左右相邻，*A* 在 *B* 左边；

```
左规则：当Rb>Ra，B的糖比A的糖数量多
右规则：当Ra>Rb，A的糖比B的糖数量多
```

相邻的学生中，评分高的学生必须获得更多的糖果 **等价于** 所有学生满足左规则且满足右规则。

步骤：

1、先给所有学生 1 颗糖

2、先从左至右遍历学生成绩 `ratings`，按照以下规则给糖，并记录在 `left` 中：

- 若R(i)>R(i-1)，则第 *i* 名学生糖比第 *i*−1 名学生多 1 个。
- 若R(i)<=R(i-1)，则第 *i* 名学生糖数量不变。（交由从右向左遍历时处理。）
- 经过此规则分配后，可以保证所有学生糖数量 **满足左规则** 。

3、同理，在此规则下从右至左遍历学生成绩并记录在 `right` 中，可以保证所有学生糖数量 **满足右规则** 。

4、最终，取以上 2 轮遍历 `left` 和 `right` 对应学生糖果数的 **最大值** ，这样则 **同时满足左规则和右规则** ，即得到每个同学的最少糖果数量。

```java
class Solution {
    public int candy(int[] ratings) {
        int[] left=new int[ratings.length];
        int[] right=new int[ratings.length];
        Arrays.fill(left,1);
        Arrays.fill(right,1);
        //左规则
        for (int i = 1; i < ratings.length; i++) {
            if (ratings[i]-ratings[i-1]>0){
                left[i]=left[i-1]+1;
            }
        }
        int count=left[ratings.length-1];
        //右规则
        for (int i = ratings.length-2; i >=0 ; i--) {
            if (ratings[i]>ratings[i+1]){
                right[i]=right[i+1]+1;
            }
            //贪心
            count+=Math.max(right[i],left[i]);
        }
        return count;
    }
}
```
