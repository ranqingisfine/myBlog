# 8.1.3 删除有序数组中的重复项

leetCode.26

**题目**：给你一个 **非严格递增排列** 的数组 `nums` ，请你**[ 原地](http://baike.baidu.com/item/原地算法)** 删除重复出现的元素，使每个元素 **只出现一次** ，返回删除后数组的新长度。元素的 **相对顺序** 应该保持 **一致** 。然后返回 `nums` 中唯一元素的个数。

考虑 `nums` 的唯一元素的数量为 `k`。去重后，返回唯一元素的数量 `k`

**分析**：

双指针，参考[移除元素](https://ranqingisfine.github.io/myBlog/%E5%85%AB%E3%80%81LeetCode/8.1%20%E6%95%B0%E7%BB%84_%E5%AD%97%E7%AC%A6%E4%B8%B2/8.1.2%20%E7%A7%BB%E9%99%A4%E5%85%83%E7%B4%A0.html)

**代码**：

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        // 创建 HashSet 用来记录已经出现过的数字，判断是否重复
        HashSet<Integer> set=new HashSet<>();
        
        // 快指针：遍历整个数组，寻找不重复的元素
        int fast=0;
        // 慢指针：指向新数组的存放位置，记录不重复元素
        int slow=0;
        
        // 快指针遍历完数组为止
        while(fast < nums.length){
            // 如果当前快指针的数字，没有出现过
            if (!set.contains(nums[fast])){
                // 把这个不重复的数字放到慢指针的位置
                nums[slow++]=nums[fast];
                // 把这个数字加入 set，标记为已出现
                set.add(nums[fast]);
            }
            // 快指针一直向后移动
            fast++;
        }
        // 慢指针的值就是去重后数组的长度
        return slow;
    }
}
```
