# 8.1.19 最后一个单词的长度

**题目**：

给你一个字符串 `s`，由若干单词组成，单词前后用一些空格字符隔开。返回字符串中 **最后一个** 单词的长度。

**单词** 是指仅由字母组成、不包含任何空格字符的最大子字符串。

**分析**：

end指针从最右往左移，直到字符不为空；start从end指针位置往左移动，直到字符为空；最后返回end-start

**代码**：

```java
class Solution {
    public int lengthOfLastWord(String s) {
        nt end=s.length()-1;
        //先判断end再判断字符的值，若反过来会下标越界异常
        while (end>=0 && s.charAt(end) == ' '){
            end--;
        }
        if (end<0){
            //若全是空字符串
            return 0;
        }
        int start=end;
        while(start>=0 && s.charAt(start) != ' '){
            start--;
        }
        return end-start;
    }
}
```
