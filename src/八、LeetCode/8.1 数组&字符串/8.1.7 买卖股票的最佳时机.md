# 8.1.7 买卖股票的最佳时机

leetCode.121

**题目**：给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

**分析**：第i天卖出股票的最大利润=第i天的股价-前面天数中最小的股价，然后与已知的最大利润比较，如果大于则更新当前最大利润的值。

**代码**：

```java
class Solution {
    //每天卖出的价格-历史最低价=获利
    public int maxProfit(int[] prices) {
        int maxProfit=0;
        int minPrice=Integer.MAX_VALUE;
        for (int i = 0; i < prices.length; i++) {
            minPrice=Math.min(minPrice,prices[i]);
            maxProfit=Math.max(prices[i]-minPrice,maxProfit);
        }
        return maxProfit;
    }
}
```
