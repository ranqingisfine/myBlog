# 8.1.24 文本左右对齐

leetCode.68

**题目**：给定一个单词数组 `words` 和一个长度 `maxWidth` ，重新排版单词，使其成为每行恰好有 `maxWidth` 个字符，且左右两端对齐的文本。

你应该使用 “**贪心算法**” 来放置给定的单词；也就是说，尽可能多地往每行中放置单词。必要时可用空格 `' '` 填充，使得每行恰好有 *maxWidth* 个字符。

要求尽可能均匀分配单词间的空格数量。如果某一行单词间的空格不能均匀分配，则左侧放置的空格数要多于右侧的空格数。

文本的最后一行应为左对齐，且单词之间不插入**额外的**空格。

**注意:**

- 单词是指由非空格字符组成的字符序列。
- 每个单词的长度大于 0，小于等于 *maxWidth*。
- 输入单词数组 `words` 至少包含一个单词。



**代码**：

```java
import java.util.ArrayList;
import java.util.List;

public class Solution {
    public List<String> fullJustify(String[] words, int maxWidth) {
        List<String> result = new ArrayList<>();
        int n = words.length;
        int index = 0; // 记录当前处理到的单词下标

        while (index < n) {
            // 1. 确定当前行能装下的单词：从index开始，找到最后一个能装入的单词end
            int totalChars = words[index].length(); // 当前行已用字符数（仅单词）
            int end = index + 1;
            while (end < n) {
                // 尝试加入下一个单词：需要额外加1个空格（单词间至少1个空格）
                if (totalChars + 1 + words[end].length() > maxWidth) {
                    break;
                }
                totalChars += 1 + words[end].length();
                end++;
            }

            // 2. 构建当前行的字符串
            StringBuilder sb = new StringBuilder();
            int wordCount = end - index; // 当前行的单词数
            int spaceCount = maxWidth - (totalChars - (wordCount - 1)); // 总需要补充的空格数（totalChars包含了wordCount-1个基础空格）

            // 情况1：最后一行 或 只有一个单词 → 左对齐，空格补在末尾
            if (end == n || wordCount == 1) {
                for (int i = index; i < end; i++) {
                    sb.append(words[i]);
                    if (i != end - 1) {
                        sb.append(" "); // 单词间仅1个空格
                    }
                }
                // 补充末尾剩余的空格
                while (sb.length() < maxWidth) {
                    sb.append(" ");
                }
            } 
            // 情况2：普通行 → 左右对齐，空格平均分配（左侧优先）
            else {
                int baseSpace = spaceCount / (wordCount - 1); // 每个间隔的基础空格数
                int extraSpace = spaceCount % (wordCount - 1); // 需要多1个空格的间隔数（前extraSpace个间隔）

                for (int i = index; i < end; i++) {
                    sb.append(words[i]);
                    if (i != end - 1) { // 不是最后一个单词，需要加空格
                        // 前extraSpace个间隔多1个空格
                        int curSpace = baseSpace + (i - index < extraSpace ? 1 : 0);
                        for (int j = 0; j < curSpace; j++) {
                            sb.append(" ");
                        }
                    }
                }
            }

            // 3. 将当前行加入结果，处理下一行
            result.add(sb.toString());
            index = end;
        }

        return result;
    }

    // 测试用例
    public static void main(String[] args) {
        Solution solution = new Solution();
        // 示例1
        String[] words1 = {"This", "is", "an", "example", "of", "text", "justification."};
        int maxWidth1 = 16;
        System.out.println(solution.fullJustify(words1, maxWidth1));
        // 输出：["This    is    an", "example  of text", "justification.  "]

        // 示例2
        String[] words2 = {"What","must","be","acknowledgment","shall","be"};
        int maxWidth2 = 16;
        System.out.println(solution.fullJustify(words2, maxWidth2));
        // 输出：["What   must   be", "acknowledgment  ", "shall be        "]

        // 示例3
        String[] words3 = {"Science","is","what","we","understand","well","enough","to","explain","to","a","computer.","Art","is","everything","else","we","do"};
        int maxWidth3 = 20;
        System.out.println(solution.fullJustify(words3, maxWidth3));
        // 输出：["Science  is  what we", "understand well enough to", "explain to  a  computer.", "Art is everything else we", "do                        "]
    }
}
```
