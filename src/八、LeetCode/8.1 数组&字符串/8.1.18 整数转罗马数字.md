# 8.1.18 整数转罗马数字

**题目**

七个不同的符号代表罗马数字，其值如下：

| 符号 | 值   |
| ---- | ---- |
| I    | 1    |
| V    | 5    |
| X    | 10   |
| L    | 50   |
| C    | 100  |
| D    | 500  |
| M    | 1000 |

罗马数字是通过添加从最高到最低的小数位值的转换而形成的。将小数位值转换为罗马数字有以下规则：

- 如果该值不是以 4 或 9 开头，请选择可以从输入中减去的最大值的符号，将该符号附加到结果，减去其值，然后将其余部分转换为罗马数字。
- 如果该值以 4 或 9 开头，使用 **减法形式**，表示从以下符号中减去一个符号，例如 4 是 5 (`V`) 减 1 (`I`): `IV` ，9 是 10 (`X`) 减 1 (`I`)：`IX`。仅使用以下减法形式：4 (`IV`)，9 (`IX`)，40 (`XL`)，90 (`XC`)，400 (`CD`) 和 900 (`CM`)。
- 只有 10 的次方（`I`, `X`, `C`, `M`）最多可以连续附加 3 次以代表 10 的倍数。你不能多次附加 5 (`V`)，50 (`L`) 或 500 (`D`)。如果需要将符号附加4次，请使用 **减法形式**。

给定一个整数，将其转换为罗马数字。



**分析**

这是解决本题最高效、最易理解的方法，核心思想是：**每次都用当前最大的可用符号去 “凑” 目标数值，直到数值减为 0**。

具体步骤：

1. 把所有罗马数字符号（包括基础和复合）按**数值从大到小**的顺序存入两个数组（数值数组 + 符号数组）。

2. 遍历数值数组，对于每个数值：

   - 只要当前整数 `num` 大于等于该数值，就把对应的符号拼接到结果中，并让 `num` 减去该数值。
   - 重复这个过程，直到 `num` 小于当前数值，再处理下一个更小的数值。

   

3. 当 `num` 减为 0 时，拼接好的字符串就是最终的罗马数字。



```java
class Solution {
    public String intToRoman(int num) {
        // 1. 定义数值数组（从大到小）
        int[] values = {1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1};
        // 2. 定义对应的罗马符号数组（与数值一一对应）
        String[] symbols = {"M", "CM", "D", "CD", "C", "XC", "L", "XL", "X", "IX", "V", "IV", "I"};
        
        // 用StringBuilder拼接字符串（效率比String高）
        StringBuilder roman = new StringBuilder();
        
        // 3. 贪心遍历：从最大的数值开始凑数
        for (int i = 0; i < values.length; i++) {
            // 只要num >= 当前数值，就拼接对应符号并减去该数值
            while (num >= values[i]) {
                roman.append(symbols[i]);
                num -= values[i];
            }
            // 提前终止循环（优化）
            if (num == 0) {
                break;
            }
        }
        
        return roman.toString();
    }
}
```
