# 8.1.17 罗马数字转整数

**题目**： 罗马数字包含以下七种字符: `I`， `V`， `X`， `L`，`C`，`D` 和 `M`。

```
字符          数值
I             1
V             5
X             10
L             50
C             100
D             500
M             1000
```

例如， 罗马数字 `2` 写做 `II` ，即为两个并列的 1 。`12` 写做 `XII` ，即为 `X` + `II` 。 `27` 写做 `XXVII`, 即为 `XX` + `V` + `II` 。

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 `IIII`，而是 `IV`。数字 1 在数字 5 的左边，所表示的数等于大数 5 减小数 1 得到的数值 4 。同样地，数字 9 表示为 `IX`。这个特殊的规则只适用于以下六种情况：

- `I` 可以放在 `V` (5) 和 `X` (10) 的左边，来表示 4 和 9。
- `X` 可以放在 `L` (50) 和 `C` (100) 的左边，来表示 40 和 90。 
- `C` 可以放在 `D` (500) 和 `M` (1000) 的左边，来表示 400 和 900。

给定一个罗马数字，将其转换成整数。



**分析**：

核心规则：

- 通常情况下，罗马数字从左到右是**大值在前，小值在后**，直接累加即可（如 VI = 5+1=6）。
- 特殊情况：当**小值出现在大值左侧**时，表示 “大值 - 小值”（如 IV = 5-1=4，IX=10-1=9，XL=50-10=40）。

我们可以通过**遍历 + 比较相邻字符**的方式解决：

1. 先建立一个 “字符 - 数值” 的映射表（比如 HashMap 或 switch 语句），方便快速查询每个罗马字符对应的数值。

2. 遍历罗马数字字符串的每个字符：

   - 如果当前字符的数值 **小于** 下一个字符的数值 → 结果中**减去**当前值（对应特殊情况）。
   - 否则 → 结果中**加上**当前值（对应常规情况）。

   

3. 遍历结束后，得到的结果就是最终的整数。



**代码**

```java
class Solution {
    public int romanToInt(String s) {
        // 1. 建立罗马字符到整数的映射表
        Map<Character, Integer> romanMap = new HashMap<>();
        romanMap.put('I', 1);
        romanMap.put('V', 5);
        romanMap.put('X', 10);
        romanMap.put('L', 50);
        romanMap.put('C', 100);
        romanMap.put('D', 500);
        romanMap.put('M', 1000);

        int result = 0;
        // 2. 遍历字符串，逐个字符处理
        for (int i = 0; i < s.length(); i++) {
            // 获取当前字符的数值
            int current = romanMap.get(s.charAt(i));
            // 3. 判断是否是特殊情况（当前值 < 下一个值）
            if (i < s.length() - 1 && current < romanMap.get(s.charAt(i + 1))) {
                result -= current; // 特殊情况：减去当前值
            } else {
                result += current; // 常规情况：加上当前值
            }
        }
        return result;
    }
    
}
```
