# 8.5.2 同构字符串

leetCode.205

**题目**：

给定两个字符串 `s` 和 `t` ，判断它们是否是同构的。

如果 `s` 中的字符可以按某种映射关系替换得到 `t` ，那么这两个字符串是同构的。

每个出现的字符都应当映射到另一个字符，同时不改变字符的顺序。不同字符不能映射到同一个字符上，相同字符只能映射到同一个字符上，字符可以映射到自己本身。

```
示例 1：
输入：s = "egg", t = "add"
输出：true
```



**分析**：

```
判断两个字符串 s 和 t 是否同构，核心规则是：
1. 两个字符串长度必须相等（代码中隐含处理，长度不等时循环不会遍历全部，但实际严谨写法会先判断长度）；
2. s 中的每个字符必须唯一映射到 t 中的一个字符（比如 e→a，则所有 e 都只能对应 a）；
3. t 中的每个字符也必须唯一映射回 s 中的一个字符（比如 a→e，则所有 a 都只能对应 e）；
简单说：映射关系必须是「一一对应」，不能是「一对多」或「多对一」。
```

**代码**：

```java
class Solution {
    public boolean isIsomorphic(String s, String t) {
        Map<Character,Character> s2t=new HashMap<>();
        Map<Character,Character> t2s=new HashMap<>();
        //题目的隐含条件是两个字符串长度要相等，加个判断更加稳妥
        if(s.length() != t.length()){
            return false;
        }
        for (int i = 0; i < s.length(); i++) {
            char sc = s.charAt(i);
            char tc = t.charAt(i);
            if (s2t.containsKey(sc) && s2t.get(sc)!=tc){
                return false;
            }
            if (t2s.containsKey(tc) && t2s.get(tc)!=sc){
                return false;
            }
            s2t.put(sc,tc);
            t2s.put(tc,sc);
        }
        return true;
    }
}
```
