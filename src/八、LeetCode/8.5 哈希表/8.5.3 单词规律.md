# 8.5.3 单词规律

leetCode.290

**题目**：给定一种规律 `pattern` 和一个字符串 `s` ，判断 `s` 是否遵循相同的规律。

这里的 **遵循** 指完全匹配，例如， `pattern` 里的每个字母和字符串 `s` 中的每个非空单词之间存在着双向连接的对应规律。具体来说：

- `pattern` 中的每个字母都 **恰好** 映射到 `s` 中的一个唯一单词。
- `s` 中的每个唯一单词都 **恰好** 映射到 `pattern` 中的一个字母。
- 没有两个字母映射到同一个单词，也没有两个单词映射到同一个字母。

```
示例：
输入: pattern = "abba", s = "dog cat cat dog"
输出: true
```



**分析**：

此题解法同：[8.5.2 同构字符串](https://ranqingisfine.github.io/myBlog/%E5%85%AB%E3%80%81LeetCode/8.5%20%E5%93%88%E5%B8%8C%E8%A1%A8/8.5.2%20%E5%90%8C%E6%9E%84%E5%AD%97%E7%AC%A6%E4%B8%B2.html)

```
- 字符和单词长度要相等
- 字符能唯一映射成单词
- 单词能唯一映射成字符
```



**代码**：

```java
class Solution {
    public boolean wordPattern(String p, String s) {
        Map<Character,String> p2s=new HashMap<>();
        Map<String,Character> s2p=new HashMap<>();
        String[] s1 = s.split(" ");
        //题目的隐含条件是长度相等，加个判断更稳妥
        if (p.length() !=s1.length){
            return false;
        }
        for (int i = 0; i < p.length(); i++) {
            char c = p.charAt(i);
            String word=s1[i];
            if (p2s.containsKey(c) && !word.equals(p2s.get(c))){
                return false;
            }
            if (s2p.containsKey(word) && s2p.get(word) != c){
                return false;
            }
            p2s.put(c,word);
            s2p.put(word,c);
        }
        return true;
    }
}
```
