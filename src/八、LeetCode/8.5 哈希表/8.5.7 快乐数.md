# 8.5.7 快乐数

leetCode.202

**题目**：编写一个算法来判断一个数 `n` 是不是快乐数。

**「快乐数」** 定义为：

- 对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和。
- 然后重复这个过程直到这个数变为 1，也可能是 **无限循环** 但始终变不到 1。
- 如果这个过程 **结果为** 1，那么这个数就是快乐数。

如果 `n` 是 *快乐数* 就返回 `true` ；不是，则返回 `false` 。

**分析**：

**用哈希集合检测循环 + 迭代计算平方和**：

1. 定义 `next` 函数：计算一个数的 “各位数字平方和”（比如 n=19 → 1²+9²=82）；
2. 用哈希集合 set记录每次计算出的平方和：
   - 如果平方和等于 1 → 是快乐数，返回 true；
   - 如果平方和已在集合中 → 说明进入循环，永远到不了 1，返回 false；
   - 否则将平方和加入集合，继续迭代计算。



**代码**：

```java
class Solution {
    public boolean isHappy(int n) {
        HashSet<Integer> set = new HashSet<>();
        while (true){
            int val=next(n);
            if (val == 1){
                return true;
            }else if (set.contains(val)){
                return false;
            }else{
                set.add(val);
                n=val;
            }
        }
    }
        public int next(int n){
        int sum=0;
        while (n != 0){
            int num=n%10;
            sum+=num*num;
            n=n/10;
        }
        return sum;
    }

}
```
