# 8.2.4 盛最多水的容器

Leetcode.11

**题目**：给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

**说明：**你不能倾斜容器。



**分析**：贪心+双指针



**代码**：

```java
class Solution {
    public int maxArea(int[] height) {
        int i=0;
        int j=height.length-1;
        int total=-1;
        while(i<j){
            total=Math.max(total,(j-i)*Math.min(height[i],height[j]));
            if (height[i]<=height[j]){
                i++;
            }else{
                j--;
            }
        }
        return total;
    }
}
```
