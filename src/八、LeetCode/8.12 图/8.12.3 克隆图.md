# 8.12.3 克隆图

Leetcode.133

**题目**：

给你无向 **[连通](https://baike.baidu.com/item/连通图/6460995?fr=aladdin)** 图中一个节点的引用，请你返回该图的 [**深拷贝**](https://baike.baidu.com/item/深拷贝/22785317?fr=aladdin)（克隆）。

图中的每个节点都包含它的值 `val`（`int`） 和其邻居的列表（`list[Node]`）。

```
class Node {
    public int val;
    public List<Node> neighbors;
}
```

题目理解：

给你一个无向图的节点： 

- 每个 node 有 val 和 neighbors 列表 
- 要求你完整克隆一张一模一样的图 
- 不能复用原来的节点，必须全部 new 新节点



**分析**：

- 用Map存新节点和旧节点的映射，避免重复创建节点

- DFS遍历原图
  - 遇到一个旧节点，先克隆出一个新节点
  - 再递归克隆出它的所有邻居，填到新节点的neighbors里

例子：

```
1 —— 2
|    |
4 —— 3
- 节点1，new一个新节点1，放进map：{旧1:新1}
- 节点2，new一个新节点2，放进map：{旧1:新1，旧2:新2}，添加节点2的邻居节点1，map里存在节点1，直接添加到节点2的neighbors里，然后添加节点2的邻居节点3
- 节点3，new一个新节点3，放进map：{旧1:新1，旧2:新2，旧3:新3}，添加节点3的邻居节点2，map里存在节点2，直接添加到节点3的neighbors里，然乎添加节点4的邻居节点4
- 节点4，new一个新节点4，放进map：{旧1:新1，旧2:新2，旧3:新3，旧4:新4}，添加节点4的邻居节点3和节点1，map里存在节点3和节点1，直接添加到neighbors里
- 节点1添加节点4，map里存在节点4，直接添加
完毕！
```



**代码**：

```java
class Solution {
    // 旧节点 -> 新节点
    private Map<Node, Node> visited = new HashMap<>();

    public Node cloneGraph(Node node) {
      	//终止条件1
        if (node == null) return null;

        // 终止条件2 已经克隆过，直接返回
        if (visited.containsKey(node)) {
            return visited.get(node);
        }

        // 操作克隆当前节点
        Node clone = new Node(node.val);
        visited.put(node, clone);

        // 节点有几个邻居就递归几次，图的深度优先搜索克隆所有邻居
        for (Node neighbor : node.neighbors) {
            clone.neighbors.add(cloneGraph(neighbor));
        }

        return clone;
    }
}
```
