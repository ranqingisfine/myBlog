# 8.12.4 除法求值

leetcode.399

## 1、题目

给你一个变量对数组 `equations` 和一个实数值数组 `values` 作为已知条件，其中 `equations[i] = [Ai, Bi]` 和 `values[i]` 共同表示等式 `Ai / Bi = values[i]` 。每个 `Ai` 或 `Bi` 是一个表示单个变量的字符串。

另有一些以数组 `queries` 表示的问题，其中 `queries[j] = [Cj, Dj]` 表示第 `j` 个问题，请你根据已知条件找出 `Cj / Dj = ?` 的结果作为答案。

返回 **所有问题的答案** 。如果存在某个无法确定的答案，则用 `-1.0` 替代这个答案。如果问题中出现了给定的已知条件中没有出现的字符串，也需要用 `-1.0` 替代这个答案。

**注意：**输入总是有效的。你可以假设除法运算中不会出现除数为 0 的情况，且不存在任何矛盾的结果。

**注意：**未在等式列表中出现的变量是未定义的，因此无法确定它们的答案。

**示例 1：**

```
输入：equations = [["a","b"],["b","c"]], values = [2.0,3.0], queries = [["a","c"],["b","a"],["a","e"],["a","a"],["x","x"]]
输出：[6.00000,0.50000,-1.00000,1.00000,-1.00000]
解释：
条件：a / b = 2.0, b / c = 3.0
问题：a / c = ?, b / a = ?, a / e = ?, a / a = ?, x / x = ?
结果：[6.0, 0.5, -1.0, 1.0, -1.0 ]
注意：x 是未定义的 => -1.0
```



**题目理解**：

给你一些除法式子：

- `a / b = 2.0`
- `b / c = 3.0`

再给你一堆查询：

- `a / c =?`
- `x / y =?`

规则：

- 能算出结果就返回值
- 算不出来（变量不存在 / 不连通）返回 `-1.0`



## 2、分析

建图+DFS找路径乘积

把问题变成图

- 每个变量=一个节点

- a/b=v等价于两条边：

  - a->b，权重为v
  - b->a，权重为1/v

  ```
  例如图：a->b->c
  若a/b=x，b/c=y，则a/c即是a->b->c等价于x*y
  ```

推导出**从x走到y，把路径上的权重乘起来**

核心在于建图和DFS

例如：

```
a / b = 2
a / d = 4
b / c = 6
求：d / c = ?
图结构如下：
     a
   /   \
  b     d
  |
  c
过程：
- cur=d，target=c，先从节点d递归到节点c，用Set<String>保证单向递归（因为建图的时候节点是双向指向的，所以若不判断已经访问的节点不再访问的话，会产生d->a->d...这种死循环）
- cur递归到target位置，return 1.0
- cur递归到节点b，return (b/c)*1.0
- cur递归到节点a，return (a/b)*(b/c)*1.0
- cur递归到节点d，return (d/a)*(a/b)*(b/c)*1.0

走死路的情况：（代码里带入这种既有死路又能成功到达的情况有助于理解）
a → b → c
↓
d
求a/c，此时从a出发有两条路，但是从d这边走是死路，return -1.0;
```

```
例如：
equations = [["a","b"],["b","c"]], values = [2.0,3.0], queries = [["a","c"],["b","a"],["a","e"],["a","a"],["x","x"]]
```



## 3、代码

```java
class Solution {
    public double[] calcEquation(List<List<String>> equations, double[] values, List<List<String>> queries) {
        //1、建图：key=节点，value=邻居:权重
        Map<String,Map<String,Double>> map=new HashMap<>();
        for (int i = 0; i < equations.size(); i++) {
            String a = equations.get(i).get(0);
            String b = equations.get(i).get(1);
            double val=values[i];
            map.computeIfAbsent(a,k->new HashMap<>()).put(b,val);
            map.computeIfAbsent(b,k->new HashMap<>()).put(a,1/val);
        }
        //2、对每个查询DFS，返回值和查询是一对一关系
        double[] res=new double[queries.size()];
        for (int i = 0; i < queries.size(); i++) {
            String start = queries.get(i).get(0);
            String end = queries.get(i).get(1);
            if (!map.containsKey(start) || !map.containsKey(end)){
                res[i]=-1;
            }else{
                res[i]=dfs(map,start,end,new HashSet<String>());
            }
        }
        return res;
    }
		//从cur 到 target，返回路径乘积
    private double dfs(Map<String,Map<String,Double>> map,String curr,String target,HashSet<String> visited){
        if (curr.equals(target)){
            return 1.0;
        }
        visited.add(curr);
        for (String key : map.get(curr).keySet()) {
            Double val = map.get(curr).get(key);
            //单向遍历
            if (!visited.contains(key)){
                double sub=dfs(map,key,target,visited);
                //这个判断的目的是有的路是死路
                if (sub != -1.0){
                    return val*sub;
                }
            }
        }
        return -1.0;

    }
}
```
