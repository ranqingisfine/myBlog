# 8.12.1 岛屿数量

leetcode.200

**题目**：

给你一个由 `'1'`（陆地）和 `'0'`（水）组成的的二维网格，请你计算网格中岛屿的数量。

岛屿总是被水包围，并且每座岛屿只能由水平方向和/或竖直方向上相邻的陆地连接形成。

此外，你可以假设该网格的四条边均被水包围。



```
题目理解：上下左右连在一起的算一个岛，求总共有多少个岛
```



**分析**：

- 遇到1，就发现了新岛，岛屿数量+1
- 马上把这座岛全部淹掉，用DFS深度优先，把这座岛所有相连的1都变成0，避免重复计数

```
例如：
行0:  1  1  0  1  0
行1:  1  0  0  1  0
行2:  1  0  1  0  0
行3:  1  1  1  1  1

- 淹完(0,0)之后的地图应该是
行0:  0  0  0  1  0
行1:  0  0  0  1  0
行2:  0  0  0  0  0
行3:  0  0  0  0  0
- 淹完(0,3)之后的地图应该是
行0:  0  0  0  0  0
行1:  0  0  0  0  0
行2:  0  0  0  0  0
行3:  0  0  0  0  0
- 所以岛屿的数量是2
```



**代码**：注意比较多是字符，而不是数字

```java
class Solution {
    public int numIslands(char[][] grid) {
        int count = 0; // 岛屿数量
        
        // 1. 遍历所有格子
        for (int i = 0; i < grid.length; i++) {
            for (int j = 0; j < grid[0].length; j++) {
                
                // 2. 发现新岛屿，注意：比较的是字符，而不是数字
                if (grid[i][j] == '1') {
                    count++;
                    // 3. 把整个岛淹掉
                    dfs(grid, i, j);
                }
            }
        }
        return count;
    }
    
    // DFS：把相连的1全变成0
    void dfs(char[][] grid, int i, int j) {
        // 越界 或者 是水 → 直接返回
        if (i < 0 || i >= grid.length || j <0 || j >= grid[0].length || grid[i][j] == '0') {
            return;
        }
        
        // 淹掉当前陆地
        grid[i][j] = '0';
        
        // 上下左右 继续淹
        dfs(grid, i-1, j); // 上
        dfs(grid, i+1, j); // 下
        dfs(grid, i, j-1); // 左
        dfs(grid, i, j+1); // 右
    }
}
```
