# 8.12.6 课程表II

leetcode.210

## 1、题目

现在你总共有 `numCourses` 门课需要选，记为 `0` 到 `numCourses - 1`。给你一个数组 `prerequisites` ，其中 `prerequisites[i] = [ai, bi]` ，表示在选修课程 `ai` 前 **必须** 先选修 `bi` 。

- 例如，想要学习课程 `0` ，你需要先完成课程 `1` ，我们用一个匹配来表示：`[0,1]` 。

返回你为了学完所有课程所安排的学习顺序。可能会有多个正确的顺序，你只要返回 **任意一种** 就可以了。如果不可能完成所有课程，返回 **一个空数组** 。

## 2、分析

此题和[[8.12.5 课程表](https://ranqingisfine.github.io/myBlog/八、LeetCode/8.12 图/8.12.5 课程表.html)]思路一模一样，在此基础上加以下逻辑

```java
发现环 → return new int[0]
递归完一个节点，把它加到 list 里
最后 list 反转，就是答案
```

为啥，因为

```
DFS 是先把后面的课修完，再回来加自己
结果顺序是：最后修的课 → 最先修的课
所以反转一下就是正常上课顺序
```



## 3、代码

```java
class Solution {
    public int[] findOrder(int numCourses, int[][] prerequisites) {
        List<List<Integer>> graph=new ArrayList<>();
        for (int i = 0; i < numCourses; i++) {
            graph.add(new ArrayList<Integer>());
        }
        for (int[] prerequisite : prerequisites) {
            int a = prerequisite[0];
            int b = prerequisite[1];
            graph.get(b).add(a);
        }
        int[] visited=new int[numCourses];
        List<Integer> array=new ArrayList<>();
        for (int i = 0; i < numCourses; i++) {
            if (dfs5(graph,visited,i,array)){
                return new int[0];
            }
        }
        Collections.reverse(array);
        int[] res=new int[array.size()];
        for (int i = 0; i < array.size(); i++) {
            res[i]=array.get(i);
        }
        return res;
    }
    private boolean dfs5(List<List<Integer>> graph,int[] visited,int curr,List<Integer> array){
        if (visited[curr] == 1){
            return true;
        }
        if (visited[curr] == 2){
            return false;
        }
        visited[curr]=1;
        for (Integer next : graph.get(curr)) {
            if (dfs5(graph,visited,next,array)){
                return true;
            }
        }
        visited[curr]=2;
        array.add(curr);
        return false;
    }
}
```
