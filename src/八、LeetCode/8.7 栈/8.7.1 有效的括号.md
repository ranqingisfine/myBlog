# 8.7.1 有效的括号

leetcode.20

**背景：介绍栈**

栈的定义

```java
//过时，线程安全但是效率低
Stack<Character> stack = new Stack<>();
//ArrayDeque替代Stack
Deque<Character> stack = new ArrayDeque<>();
```

`ArrayDeque` 既可以当「栈」用（后进先出），也可以当「队列」用（先进先出）

| 场景         | 方法名                  | 作用                                | 返回值 / 异常                                 | 对应 Stack 方法   |
| ------------ | ----------------------- | ----------------------------------- | --------------------------------------------- | ----------------- |
| 作为栈使用   | `push(E e)`             | 元素压入「栈顶」（等价于 addFirst） | 无返回值；不抛异常                            | `Stack.push()`    |
|              | `pop()`                 | 移除并返回「栈顶」元素              | 返回栈顶元素；空时抛 `NoSuchElementException` | `Stack.pop()`     |
|              | `peek()`                | 查看「栈顶」元素（不移除）          | 返回栈顶元素；空时抛 `NoSuchElementException` | `Stack.peek()`    |
| 通用操作     | `isEmpty()`             | 判断是否为空                        | boolean（空返回 true）                        | `Stack.isEmpty()` |
|              | `size()`                | 返回元素数量                        | int                                           | `Stack.size()`    |
|              | `clear()`               | 清空所有元素                        | void                                          | `Stack.clear()`   |
| 作为队列使用 | `add(E e)`/`offer(E e)` | 元素添加到「队列尾部」              | add 抛异常，offer 返回 boolean                | -                 |
|              | `remove()`/`poll()`     | 移除并返回「队列头部」              | remove 抛异常，poll 返回 null                 | -                 |



**题目**：

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

```
示例 1：
输入：s = "()"
输出：true
```





**分析**：

- 哈希表建立右括号和左括号的映射关系
- 如果遇到右括号，就出栈判断栈顶元素是否正确，然后移除栈顶元素
- 最后若栈为空，则返回true



**代码**：

```java
class Solution {
    public boolean isValid(String s) {
        int n=s.length();
        if(n%2 == 1){
            return false;
        }
        Map<Character,Character> pairs=new HashMap<>();
        pairs.put(')','(');
        pairs.put('}','{');
        pairs.put(']','[');
        Deque<Character> stack=new ArrayDeque<>();
        for(int i=0;i<n;i++){
            char ch=s.charAt(i);
            if(pairs.containsKey(ch)){
                if(stack.isEmpty() || stack.peek()!=pairs.get(ch)){
                    return false;
                }
                stack.pop();
            }else{
                stack.push(ch);
            }
        }
        return stack.isEmpty();
    }
}
```
