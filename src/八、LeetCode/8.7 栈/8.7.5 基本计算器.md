# 8.7.5 基本计算器

leetCode.224

**题目**：

给你一个字符串表达式 `s` ，请你实现一个基本计算器来计算并返回它的值。

注意:不允许使用任何将字符串作为数学表达式计算的内置函数，比如 `eval()` 。

```
示例 3：
输入：s = "(1+(4+5+2)-3)+(6+8)"
输出：23
```



**分析**：

例如计算：12+21+(3-2)

- 无括号加减：用 `sign` 标记正负，把减法转成 “加负数”，只用 `res += sign×数字`；

- 左括号 `(`：把当前的 `res` 和 `sign` 存进栈，然后重置 `res=0、sign=1`，算括号里的；

- 右括号 `)`：从栈里取出之前存的 `sign` 和 `res`，合并成 `prevRes + prevSign×当前res`；

- 细节：跳过空格，多位数要拼接，用 long 防溢出。



**代码**：

```java
class Solution {
    public int calculate(String s) {
        Deque<Integer> stack = new ArrayDeque<>(); // 存储括号前的结果和符号
        int res = 0; // 当前计算结果
        int sign = 1; // 当前符号：1为正，-1为负
        int n = s.length();
        
        for (int i = 0; i < n; i++) {
            char c = s.charAt(i);
            if (c == ' ') {
                // 跳过空格
                continue;
            } else if (c == '+') {
                // 更新符号为正
                sign = 1;
            } else if (c == '-') {
                // 更新符号为负
                sign = -1;
            } else if (c == '(') {
                // 遇到左括号：压入当前结果和符号，重置res和sign
                stack.push(res);
                stack.push(sign);
                res = 0;
                sign = 1;
            } else if (c == ')') {
                // 遇到右括号：弹出符号和结果，合并计算
                int prevSign = stack.pop();
                int prevRes = stack.pop();
                res = prevRes + prevSign * res;
            } else if (Character.isDigit(c)) {
                // 处理连续数字（比如"123"）
                long num = 0; // 用long避免溢出
                while (i < n && Character.isDigit(s.charAt(i))) {
                    num = num * 10 + (s.charAt(i) - '0');
                    i++;
                }
                i--; // 修正for循环的i++，避免跳过下一个字符
                // 累加当前数字（乘以符号）
                res += sign * num;
            }
        }
        return res;
    }
}
```
