# 8.7.4 逆波兰表达式求值

leetCode.150

**题目**：

给你一个字符串数组 `tokens` ，表示一个根据 [逆波兰表示法](https://baike.baidu.com/item/逆波兰式/128437) 表示的算术表达式。

请你计算该表达式。返回一个表示表达式值的整数。



**分析**：

逆波兰表达式（后缀表达式）是把运算符写在操作数后面的表达式，特点是：

1. 无需括号就能表达运算优先级；
2. 计算规则：遇到数字直接入栈，遇到运算符则弹出栈顶两个数字，用**后弹出的数** 对 **先弹出的数** 执行运算，结果入栈；
3. 最终栈中只剩一个数，就是表达式的结果。

示例：`["2","1","+","3","*"]` → 等价于 `(2+1)*3=9`，计算过程：2、1 入栈 → 遇到 `+`，弹出 1、2 → 2+1=3 入栈 → 3 入栈 → 遇到 `*`，弹出 3、3 → 3*3=9 入栈 → 最终结果 9。



**代码**：

```java
class Solution {
    public int evalRPN(String[] tokens) {
        Deque<Integer> stack=new LinkedList<>();;
        int length = tokens.length;
        for (int i = 0; i < length; i++) {
            //如果是数字，直接压入栈
            if (isNumber(tokens[i])){
                stack.push(Integer.parseInt(tokens[i]));
            }else{
                //如果不是数字，则出栈并计算
                Integer x = stack.pop();
                Integer y = stack.pop();
                switch (tokens[i]){
                    case "+":
                        stack.push(x+y);
                        break;
                    case "-":
                    		//注意弹出顺序
                        stack.push(y-x);
                        break;
                    case "*":
                        stack.push(x*y);
                        break;
                    case "/":
                    		//注意弹出顺序
                        stack.push(y/x);
                        break;
                }
            }
        }
        return stack.pop();
    }

        public  boolean isNumber(String str){
        if ("+".equals(str)||"-".equals(str)||"*".equals(str)||"/".equals(str)){
            return false;
        }
        return true;
    }
}
```
