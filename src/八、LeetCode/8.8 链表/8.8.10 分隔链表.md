# 8.8.10 分隔链表

leetCode.86

**题目**：

给你一个链表的头节点 `head` 和一个特定值 `x` ，请你对链表进行分隔，使得所有 **小于** `x` 的节点都出现在 **大于或等于** `x` 的节点之前。

你应当 **保留** 两个分区中每个节点的初始相对位置。



**分析**：

分两队 + 合并



**代码**：

```java
// 链表节点定义（题目已给出）
class ListNode {
    int val;
    ListNode next;
    ListNode() {}
    ListNode(int val) { this.val = val; }
    ListNode(int val, ListNode next) { this.val = val; this.next = next; }
}

class Solution {
    public ListNode partition(ListNode head, int x) {
        // 1. 初始化两个虚拟头节点（管理两部分链表）
        ListNode dummy1 = new ListNode(0); // 小于x的链表头
        ListNode dummy2 = new ListNode(0); // 大于等于x的链表头
        // 2. 尾指针：用于拼接节点（始终指向各队的最后一个节点）
        ListNode p1 = dummy1; // 队1尾指针
        ListNode p2 = dummy2; // 队2尾指针
        // 3. 遍历原链表，分两队
        ListNode curr = head;
        while (curr != null) {
            if (curr.val < x) {
                // 归到队1：拼到p1后面，p1后移
                p1.next = curr;
                p1 = p1.next;
            } else {
                // 归到队2：拼到p2后面，p2后移
                p2.next = curr;
                p2 = p2.next;
            }
            // 关键：curr后移（继续遍历原链表）
            curr = curr.next;
        }
        // 4. 合并两队：队1尾 → 队2头
        p1.next = dummy2.next;
        // 5. 处理队2尾：避免成环（队2最后一个节点的next必须置null）
        p2.next = null;
        // 6. 返回队1的头（dummy1.next）
        return dummy1.next;
    }
}ß
```
