# 8.8.11 LRU缓存

leetCode.146

**题目**：

请你设计并实现一个满足 [LRU (最近最少使用) 缓存](https://baike.baidu.com/item/LRU) 约束的数据结构。

实现 `LRUCache` 类：

- `LRUCache(int capacity)` 以 **正整数** 作为容量 `capacity` 初始化 LRU 缓存
- `int get(int key)` 如果关键字 `key` 存在于缓存中，则返回关键字的值，否则返回 `-1` 。
- `void put(int key, int value)` 如果关键字 `key` 已经存在，则变更其数据值 `value` ；如果不存在，则向缓存中插入该组 `key-value` 。如果插入操作导致关键字数量超过 `capacity` ，则应该 **逐出** 最久未使用的关键字。

函数 `get` 和 `put` 必须以 `O(1)` 的平均时间复杂度运行。



**分析**：

哈希表+双向链表

执行 `get(key)`：

✅ 存在：找到节点 → 删除该节点 → 插入到表头（标记为最近使用）→ 返回值；

❌ 不存在：返回 -1；

执行 `put(key, value)`：

✅ 存在：找到节点 → 更新值 → 删除该节点 → 插入到表头；

❌ 不存在：

- 若缓存满：删除尾节点（LRU）→ 删除哈希表中对应键 → 创建新节点 → 插入表头 → 哈希表添加键值对；
- 若缓存未满：创建新节点 → 插入表头 → 哈希表添加键值对。



**代码**：

```java
import java.util.HashMap;
import java.util.Map;

// 双向链表节点定义
class DLinkedNode {
    int key;
    int value;
    DLinkedNode prev;
    DLinkedNode next;
    public DLinkedNode() {}
    public DLinkedNode(int _key, int _value) {
        key = _key;
        value = _value;
    }
}

class LRUCache {
    // 哈希表：key -> 双向链表节点
    private Map<Integer, DLinkedNode> cache = new HashMap<>();
    // 缓存容量
    private int capacity;
    // 虚拟头/尾节点（简化边界处理）
    private DLinkedNode head, tail;

    // 初始化LRU缓存
    public LRUCache(int capacity) {
        this.capacity = capacity;
        // 初始化虚拟头、尾节点，互相指向
        head = new DLinkedNode();
        tail = new DLinkedNode();
        head.next = tail;
        tail.prev = head;
    }

    // 获取键对应的值
    public int get(int key) {
        DLinkedNode node = cache.get(key);
        if (node == null) {
            return -1; // 键不存在
        }
        // 访问后，将节点移到表头（标记为最近使用）
        moveToHead(node);
        return node.value;
    }

    // 插入/更新键值对
    public void put(int key, int value) {
        DLinkedNode node = cache.get(key);
        if (node == null) {
            // 键不存在：创建新节点
            DLinkedNode newNode = new DLinkedNode(key, value);
            // 哈希表添加映射
            cache.put(key, newNode);
            // 插入到表头
            addToHead(newNode);
            // 检查容量：超过则淘汰LRU节点（尾节点前驱）
            if (cache.size() > capacity) {
                DLinkedNode tailNode = removeTail();
                // 哈希表删除对应键
                cache.remove(tailNode.key);
            }
        } else {
            // 键存在：更新值，移到表头
            node.value = value;
            moveToHead(node);
        }
    }

    // 辅助方法1：将节点插入到虚拟头节点之后（表头）
    private void addToHead(DLinkedNode node) {
        node.prev = head;
        node.next = head.next;
        head.next.prev = node;
        head.next = node;
    }

    // 辅助方法2：删除双向链表中的任意节点
    private void removeNode(DLinkedNode node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }

    // 辅助方法3：将节点移到表头（先删除，再插入）
    private void moveToHead(DLinkedNode node) {
        removeNode(node);
        addToHead(node);
    }

    // 辅助方法4：删除尾节点（LRU节点），返回该节点
    private DLinkedNode removeTail() {
        DLinkedNode res = tail.prev;
        removeNode(res);
        return res;
    }
}

/**
 * Your LRUCache object will be instantiated and called as such:
 * LRUCache obj = new LRUCache(capacity);
 * int param_1 = obj.get(key);
 * obj.put(key,value);
 */
```
