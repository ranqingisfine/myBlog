# 8.8.9 旋转链表

leetCode.61

**题目**：

给你一个链表的头节点 `head` ，旋转链表，将链表每个节点向右移动 `k` 个位置。



**分析**：

将链表分为两部分，重新拼接即可，注意判断空



**代码**：

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode rotateRight(ListNode head, int k) {
        // 修复1：提前处理空链表/单节点链表（边界优化）
        if (head == null || head.next == null) {
            return head;
        }

        ListNode dummy = new ListNode(0);
        dummy.next = head;
        ListNode curr = head;
        int len = 0;
        ListNode end = null; // 链表尾节点

        // 1. 计算链表长度，找到尾节点end
        while (curr != null) {
            len++;
            if (curr.next == null) {
                end = curr; // 记录尾节点
            }
            curr = curr.next;
        }

        // 2. 计算有效旋转步数（k可能大于len，取模）
        k = k % len;
        if (k == 0) { // 旋转len次等于没旋转
            return head; // 直接返回原头，无需操作
        }

        // 3. 找到旋转分割点的前驱节点pre（走len-k步）
        ListNode pre = dummy;
        for (int i = 0; i < len - k; i++) {
            pre = pre.next;
        }

        // 4. 核心旋转逻辑：拼接链表
        ListNode oldHead = dummy.next; // 原链表头（优化命名）
        dummy.next = pre.next; // 新链表头 = 分割点的下一个节点
        end.next = oldHead;    // 尾节点指向原头，形成环后断开
        pre.next = null;       // 断开分割点，完成旋转

        return dummy.next;
    }
}
```
