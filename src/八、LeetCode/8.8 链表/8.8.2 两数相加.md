# 8.8.2 两数相加

leetCode.2

**题目**：

给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。



**代码**：

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        // 1. 初始化：虚拟头节点（简化头节点处理）、进位（初始0）、结果链表的尾指针
        ListNode dummy = new ListNode(0); // 虚拟头节点，值无意义
        ListNode curr = dummy; // 尾指针，用于拼接新节点
        int carry = 0; // 进位，初始为0

        // 2. 遍历：只要有链表没走完，或还有进位，就继续
        while (l1 != null || l2 != null || carry != 0) {
            // 3. 取当前位的值（空节点取0）
            int val1 = (l1 != null) ? l1.val : 0;
            int val2 = (l2 != null) ? l2.val : 0;

            // 4. 计算当前位总和 + 进位
            int sum = val1 + val2 + carry;
            int currentVal = sum % 10; // 当前位的结果（取个位）
            carry = sum / 10; // 新的进位（取十位）

            // 5. 构建结果节点，挂到尾指针后
            curr.next = new ListNode(currentVal);
            curr = curr.next; // 尾指针后移

            // 6. 原链表指针后移（不为空时）
            if (l1 != null) l1 = l1.next;
            if (l2 != null) l2 = l2.next;
        }

        // 7. 返回结果链表的真正头节点（虚拟头节点的下一个）
        return dummy.next;
    }
}
```
