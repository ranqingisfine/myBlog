# 8.8.3 合并两个有序链表

leetCode.21

**题目**：

将两个升序链表合并为一个新的 **升序** 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。



**分析**：

- 初始化虚拟头节点
- 双指针遍历两个有序链表
- 比较当前的值，选更小的拼到结果里
- 处理剩余节点
- 返回真正的头节点



**代码**：

```java
// 链表节点定义（题目已给出）
class ListNode {
    int val;
    ListNode next;
    ListNode() {}
    ListNode(int val) { this.val = val; }
    ListNode(int val, ListNode next) { this.val = val; this.next = next; }
}

class Solution {
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        // 1. 初始化虚拟头节点（简化头节点处理）、结果链表尾指针
        ListNode dummy = new ListNode(0); // 虚拟头节点，值无意义
        ListNode curr = dummy; // 尾指针，用于拼接节点
        
        // 2. 双指针遍历两个有序链表
        ListNode p1 = list1; // 指向list1当前节点
        ListNode p2 = list2; // 指向list2当前节点
      //p1/p2 是「临时遍历指针」，目的是保护原始的 list1/list2 引用不被修改（虽然题目中不需要，但这是工程开发的好习惯）。
        while (p1 != null && p2 != null) {
            // 3. 比较当前节点值，选更小的拼到结果里
            if (p1.val <= p2.val) {
                curr.next = p1; // 拼接p1
                p1 = p1.next;   // p1后移
            } else {
                curr.next = p2; // 拼接p2
                p2 = p2.next;   // p2后移
            }
            curr = curr.next; // 尾指针后移，指向新的尾部
        }
        
        // 4. 处理剩余节点（其中一个链表已遍历完）
        if (p1 != null) {
            curr.next = p1; // 拼接p1剩余部分
        } else {
            curr.next = p2; // 拼接p2剩余部分
        }
        
        // 5. 返回结果链表的真正头节点
        return dummy.next;
    }
}
```
