# 8.14.2 添加与搜索单词-数据结构设计

Leetcode.211

## 1、题目

请你设计一个数据结构，支持 添加新单词 和 查找字符串是否与任何先前添加的字符串匹配 。

实现词典类 `WordDictionary` ：

- `WordDictionary()` 初始化词典对象
- `void addWord(word)` 将 `word` 添加到数据结构中，之后可以对它进行匹配
- `bool search(word)` 如果数据结构中存在字符串与 `word` 匹配，则返回 `true` ；否则，返回 `false` 。`word` 中可能包含一些 `'.'` ，每个 `.` 都可以表示任何一个字母。



## 2、分析

- 前缀树结构和添加跟[8.14.1 实现Trie（前缀树）](https://ranqingisfine.github.io/myBlog/%E5%85%AB%E3%80%81LeetCode/8.14%20%E5%AD%97%E5%85%B8%E6%A0%91/8.14.1%20%E5%AE%9E%E7%8E%B0Trie%EF%BC%88%E5%89%8D%E7%BC%80%E6%A0%91%EF%BC%89.html)一样
- 普通查询也一样
- **遇到 `.` 时要做 DFS 回溯**
  - 对当前节点的 **26 个孩子都试一遍**
  - 只要有一条路能走到底，就返回 `true`

这里再次强调下，dfs是先走到最后一个节点再回溯



## 3、代码

```java
class WordDictionary {

    class TrieNode {
        TrieNode[] children;
        boolean isEnd;

        public TrieNode() {
            children = new TrieNode[26];
            isEnd = false;
        }
    }

    private TrieNode root;

    public WordDictionary() {
        root = new TrieNode();
    }

    // 添加单词，和 208 完全一样
    public void addWord(String word) {
        TrieNode cur = root;
        for (char c : word.toCharArray()) {
            int idx = c - 'a';
            if (cur.children[idx] == null) {
                cur.children[idx] = new TrieNode();
            }
            cur = cur.children[idx];
        }
        cur.isEnd = true;
    }

    // 搜索入口
    public boolean search(String word) {
        return dfs(root, word, 0);
    }

    // DFS 递归搜索
    // node: 当前节点
    // word: 要搜索的单词
    // index: 正在匹配第几个字符
    private boolean dfs(TrieNode node, String word, int index) {
        // 递归出口：单词全部匹配完
        if (index == word.length()) {
            return node.isEnd;
        }

        char c = word.charAt(index);

        // 情况1：不是点，正常走
        if (c != '.') {
            int idx = c - 'a';
            // 没路就 false
            if (node.children[idx] == null) {
                return false;
            }
            // 继续匹配下一个字符
            return dfs(node.children[idx], word, index + 1);
        }

        // 情况2：是点，遍历所有 26 个孩子
        for (int i = 0; i < 26; i++) {
            // 有孩子就试着走
            if (node.children[i] != null) {
                // 只要有一条走通，直接返回 true
                if (dfs(node.children[i], word, index + 1)) {
                    return true;
                }
            }
        }

        // 所有孩子都走不通
        return false;
    }
}
```
