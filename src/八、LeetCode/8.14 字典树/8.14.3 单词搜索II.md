# 8.14.3 单词搜索II

leetcode.212

## 1、题目

给定一个 `m x n` 二维字符网格 `board` 和一个单词（字符串）列表 `words`， *返回所有二维网格上的单词* 。

单词必须按照字母顺序，通过 **相邻的单元格** 内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母在一个单词中不允许被重复使用。

## 2、分析

核心一句话：**Trie + DFS 回溯**

用前缀树把要找的单词存起来，再在棋盘上每个起点 DFS，沿着 Trie 走，找到就收集。

**把所有单词建成一棵 Trie**

方便快速判断：当前路径是不是某个单词的前缀，是不是完整单词。

**在棋盘上逐格 DFS**

- 从每个格子出发，向上下左右四个方向走
- 每走一步，都沿着 Trie 往下查
- 走到 Trie 标记 `isEnd` 的节点，说明找到一个单词
- 用 `Set` 去重，避免重复答案

**回溯**

走过的格子临时标记，避免回头，递归回来再恢复。

例如：

```java
char[][] board = {
    {'o','a','a','n'},
    {'e','t','a','e'},
    {'i','h','k','r'},
    {'i','f','l','v'}
};

String[] words = {"oath", "pea", "eat", "rain"};
```

演示从(0,0)出发，找到oath的过程

```
1.把单词建成前缀树
root
├─ o → a → t → h (word=oath)
├─ p → e → a (word=pea)
├─ e → a → t (word=eat)
└─ r → a → i → n (word=rain)
每个单词结尾的节点，我们存上 word = 字符串。
2.遍历棋盘每个棋子，逐个DFS
	- 从(0,0)处开始，往上/左走不通，往下走的e不在前缀树的下个节点里面，也走不通，只有往右走的'a'能通，所以往右走
	- 到达(0,1)位置后，只有下面走的t符合要求，所以往下走
	- 到达(1,1)位置后，同样只有下面走的h符合要求，所以向下走
	- 到达(2,1)位置后，当前前缀树节点没有下个节点了，所以返回
```

前缀树控制了DFS遍历的方向

## 3、代码

```java
class Solution {
    // Trie 节点
    class TrieNode {
        TrieNode[] children = new TrieNode[26];
        String word = null; // 存完整单词，找到直接拿
    }

    // 建 Trie
    private TrieNode buildTrie(String[] words) {
        TrieNode root = new TrieNode();
        for (String w : words) {
            TrieNode cur = root;
            for (char c : w.toCharArray()) {
                int idx = c - 'a';
                if (cur.children[idx] == null)
                    cur.children[idx] = new TrieNode();
                cur = cur.children[idx];
            }
            cur.word = w; // 单词结尾存单词本身
        }
        return root;
    }

    public List<String> findWords(char[][] board, String[] words) {
        TrieNode root = buildTrie(words);
        Set<String> res = new HashSet<>(); // 去重
        int m = board.length;
        int n = board[0].length;

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                dfs(board, i, j, root, res);
            }
        }
        return new ArrayList<>(res);
    }

    // 从 (i,j) 开始，在 Trie 的 node 节点往下搜
    private void dfs(char[][] board, int i, int j, TrieNode node, Set<String> res) {
        // 越界
        if (i < 0 || i >= board.length || j < 0 || j >= board[0].length) return;

        char c = board[i][j];
        // 已访问 或 Trie 没路
        if (c == '#' || node.children[c - 'a'] == null) return;

        // 走到下一层 Trie 节点
        node = node.children[c - 'a'];

        // 找到一个单词
        if (node.word != null) {
            res.add(node.word);
            // 这里可以不 return，因为可能有更长单词：oath -> oathful
        }

        // 标记已访问
        board[i][j] = '#';

        // 四个方向
        dfs(board, i + 1, j, node, res);
        dfs(board, i - 1, j, node, res);
        dfs(board, i, j + 1, node, res);
        dfs(board, i, j - 1, node, res);

        // 回溯恢复
        board[i][j] = c;
    }
}
```
