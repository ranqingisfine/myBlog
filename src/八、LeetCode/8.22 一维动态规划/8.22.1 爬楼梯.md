# 8.22.1 爬楼梯

leetcode.70

## 1、斐波那契数列

什么是斐波那契数列？

**从第 3 个数开始，每个数 = 前两个数之和**

最常见的开头是：

```
1, 1, 2, 3, 5, 8, 13, 21, 34...
```

公式：fib(n) = fib(n-1) + fib(n-2)

题目：求斐波那契数列

方法1：递归（缺点是效率太低）

```java
public static int fibonacci(int n){
        if (n == 1 || n == 2) {             //特殊情况，分开讨论
            return 1;
        }
        if (n > 2) {
            return fibonacci(n - 1) + fibonacci(n - 2);     //递归调用
        }
        return -1;              //如果输入错误的n，一律返回-1
}
```

方法2：for循环（推荐）

```java
class Solution {
    public int fib(int n) {
        // dp[i] = 第i个斐波那契数，因为数组下标是从0开始的，所以第n个斐波那契数对应的下标是n+1
        int[] dp = new int[n + 1];
        // 前两个固定
        if (n >= 0) dp[0] = 1; 
        if (n >= 1) dp[1] = 1;
        // 后面每个 = 前两个之和
        for (int i = 2; i <= n; i++) {
            dp[i] = dp[i-1] + dp[i-2];
        }
        return dp[n];
    }
}
```



## 2、题目

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？



## 3、分析

要爬到第 `n` 阶，只有两种可能：

1. 从 `n-1` 阶 **爬 1 阶** 上来
2. 从 `n-2` 阶 **爬 2 阶** 上来

所以：

**f(n) = f(n-1) + f(n-2)**

这不就是**斐波那契数列**吗！



## 4、代码

```java
class Solution {
    public int climbStairs(int n) {
        // dp[i] 表示爬到第 i 级台阶的总方法数
        int[] dp = new int[n + 1];
        // 初始状态（边界条件）
        if (n >= 1) dp[1] = 1; // 1级台阶：1种方法
        if (n >= 2) dp[2] = 2; // 2级台阶：2种方法

        // 动态规划递推公式
        // 爬到第i级 = 最后一步爬1级(dp[i-1]) + 最后一步爬2级(dp[i-2])
        for (int i = 3; i <= n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
}
```
