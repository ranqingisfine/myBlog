# 8.22.5 最长递增子序列

Leetcode.300

## 1、题目

给你一个整数数组 `nums` ，找到其中最长严格递增子序列的长度。

**子序列** 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，`[3,6,2,7]` 是数组 `[0,3,1,6,2,2,7]` 的子序列。

**示例 1：**

```
输入：nums = [10,9,2,5,3,7,101,18]
输出：4
解释：最长递增子序列是 [2,3,7,101]，因此长度为 4 。
```



## 2、分析

dp[i]：以nume[i]这个数结尾的最长递增子序列长度，每个数自己就是一个序列，所以初始全部为1。

转移方程：对每个i，遍历它前面所有j，如果nums[j]<nums[i]，那么dp[i]=max(dp[i],dp[j]+1)

理解：前面有个更小的数，我就可以接在它后面，长度+1

答案：整个dp数组里的最大值

```
例：
nums=[10,9,2,5,3,7,101,18]
       0,1,2,3,4,5,6,  7
当i=3，j=2时，dp[3]=max(dp[3],dp[2]+1)=2
理解：当i=3时，j从0～2取值，如果dp[j]<dp[3]，那么就把dp[3]接在这个dp[j]的后面，即是dp[3]=dp[j+1]了
```



## 3、代码

```java
public int lengthOfLIS(int[] nums) {
    int n = nums.length;
    int[] dp = new int[n];
    // 初始全部为 1
    Arrays.fill(dp, 1);

    // 遍历每个结尾 i
    for (int i = 0; i < n; i++) {
        // 看 i 前面所有 j
        for (int j = 0; j < i; j++) {
            // 能接就更新
            if (nums[j] < nums[i]) {
                dp[i] = Math.max(dp[i], dp[j] + 1);
            }
        }
    }

    // 找 dp 里最大的
    int maxLen = 0;
    for (int x : dp) {
        maxLen = Math.max(maxLen, x);
    }
    return maxLen;
}
```
