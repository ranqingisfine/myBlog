# 8.22.4 零钱兑换

leetcode.322

  ## 1、题目

给你一个整数数组 `coins` ，表示不同面额的硬币；以及一个整数 `amount` ，表示总金额。

计算并返回可以凑成总金额所需的 **最少的硬币个数** 。如果没有任何一种硬币组合能组成总金额，返回 `-1` 。

你可以认为每种硬币的数量是无限的。

## 2、分析

核心思路：要凑出金额i的最少硬币数=在所有硬币coin中，取凑出i-coin的最少个数+1里最小的那个

dp[i]：凑成总金额i所需要的最少硬币个数，数组长度，amount+1

初始化：dp[0]=0，凑0元，需要0个硬币；其余dp[i]，初始化为一个很大的数，比如amount+1，表示暂时凑不出来

对每个金额i，遍历每个硬币coin，如果coin<=i，则dp[i]=min(dp[i],dp[i-coin]+1)；理解：我用这枚硬币，剩下的钱就是i-coin它的最少个数+1，就是当前方案的硬币数，所有方案里最小的

最终答案：如果dp[amount]还是很大-->凑不出来，返回-1；否则返回dp[amount]；

```
例如：coins[1,2,5]  dp[0]=0
amount=1时，dp[1]=Min(dp[1],dp[1-1]+1)=1
amount=2时，dp[2]=Min(dp[2],dp[2-1]+1)=2;dp[2]=Min(dp[2],dp[2-2]+1)=1 ---> dp[2]=1
```



## 3、代码

```java
class Solution {
    public int coinChange(int[] coins, int amount) {
        int max=amount+1;
        int[] dp=new int[amount+1];
        Arrays.fill(dp,max);
        dp[0]=0;
        for (int i = 1; i <= amount; i++) {
            for (int coin : coins) {
                if(i>=coin){
                    dp[i]=Math.min(dp[i],dp[i-coin]+1);
                }
            }
        }
        return dp[amount] == max ? -1 : dp[amount];
    }
}
```
