# 8.20.4 只出现一次的数字

Leetcode.136

## 1、题目

给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

**示例 1 ：**

**输入：**nums = [2,2,1]

**输出：**1

**示例 2 ：**

**输入：**nums = [4,1,2,1,2]

**输出：**4

## 2、分析

利用 **异或运算（XOR）** 的性质：

1. 相同数字异或 = 0

   

   ```
   a ^ a = 0
   ```

2. 0 异或任何数 = 那个数本身

   

   ```
   0 ^ a = a
   ```

3. 异或满足交换律、结合律

所以：

**把数组里所有数字全部异或一遍，最后剩下的就是只出现一次的那个数！**

```
举个例子：数组：[1,2,1,3,2]
全部异或：1 ^ 2 ^ 1 ^ 3 ^ 2
交换顺序：(1 ^ 1) ^ (2 ^ 2) ^ 3= 0 ^ 0 ^ 3= 3
成对的都抵消成 0，只剩单独的那个。
```



## 3、代码

```java
class Solution {
    public int singleNumber(int[] nums) {
        int res = 0;
        for (int num : nums) {
            res ^= num;
        }
        return res;
    }
}
```
