# 8.20.2 颠倒二进制位

leetcode.190

## 1、题目

颠倒给定的 32 位有符号整数的二进制位。



## 2、分析

位运算基础：

- n & 1 

```
作用：只拿二进制的最后一位
比如：n 二进制：10110最后一位是 0
n & 1 = 0
n 二进制：10111最后一位是 1n & 1 = 1
```

- res <<= 1

```
作用：二进制整体往左挪一位，最右边补 0（给新数腾位置）
比如 res 现在是：101左移一位变成：1010
```

- res |= 某数

```
作用：把最后一位变成 0 或 1
比如 res 现在是：1010想把最后一位变成 1
res |= 1 → 1011
```

因此，这题解决步骤

- 结果左移一位，空出位置
- 把 n 的最后一位加到 res 末尾
- n 右移一位，处理下一位

## 3、代码

```java
public class Solution {
    // you need treat n as an unsigned value
    public int reverseBits(int n) {
        int res = 0;
        for (int i = 0; i < 32; i++) {
            // 结果左移一位，空出位置
            res <<= 1;
            // 把 n 的最后一位加到 res 末尾
            res |= n & 1;
            // n 右移一位，处理下一位
            n >>= 1;
        }
        return res;
    }
}
```
