# 8.20.1 二进制求和

leetcode.67

## 1、题目

给你两个二进制字符串 `a` 和 `b` ，以二进制字符串的形式返回它们的和。

**示例 1：**

```
输入:a = "11", b = "1"
输出："100"
```

**示例 2：**

```
输入：a = "1010", b = "1011"
输出："10101"
```

## 2、分析

**从后往前扫**

**三位相加：两个数字 + 一个进位**

**进位 = sum / 2**

**本位 = sum % 2**

**最后反转字符串**



## 3、代码

```java
class Solution {
    public String addBinary(String a, String b) {
        StringBuilder sb = new StringBuilder();
        int i = a.length() - 1;
        int j = b.length() - 1;
        int carry = 0; // 进位

        while (i >= 0 || j >= 0 || carry > 0) {
            // 取当前位，没数了就取0
            int digitA = i >= 0 ? a.charAt(i--) - '0' : 0;
            int digitB = j >= 0 ? b.charAt(j--) - '0' : 0;

            // 求和
            int sum = digitA + digitB + carry;
            
            // 算新进位
            carry = sum / 2;
            
            // 算当前位（取余数）
            sb.append(sum % 2);
        }

        // 因为是从后往前算的，结果要反转
        return sb.reverse().toString();
    }
}
```
