# 8.20.6 数字范围按与

leetcode.201

## 1、题目

给你两个整数 `left` 和 `right` ，表示区间 `[left, right]` ，返回此区间内所有数字 **按位与** 的结果（包含 `left` 、`right` 端点）。

**示例 1：**

```
输入：left = 5, right = 7
输出：4
```



## 2、分析

一句话结论：

```
区间 [left, right] 所有数字按位与的结果，就是 left 和 right 二进制的最长公共前缀，后面全部补 0。
```

例子：

```
left = 5（101），right = 7（111）
5≠7 → 都右移 1 位：2 (10)、3 (11)
2≠3 → 再右移 1 位：1 (1)、1 (1)
相等了，公共前缀是 1
左移 2 位 → 100 → 4
验证：5 & 6 & 7 = 4 ✅
```

解法：

- 把 left、right **一起右移**，直到两者相等

- 相等时就是**最长公共前缀**

- 再把结果**左移回去**（补 0）



## 3、代码

```java
class Solution {
    public int rangeBitwiseAnd(int left, int right) {
        int shift = 0;
        // 找公共前缀
        while (left != right) {
            left >>= 1;
            right >>= 1;
            shift++;
        }
        // 左移补回 0
        return left << shift;
    }
}
```
