# 8.3.4 最小覆盖子串

**题目**：给定两个字符串 `s` 和 `t`，长度分别是 `m` 和 `n`，返回 s 中的 **最短窗口 子串**，使得该子串包含 `t` 中的每一个字符（**包括重复字符**）。如果没有这样的子串，返回空字符串 `""`。

测试用例保证答案唯一。

**示例 1：**

```
输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
解释：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
```



**分析**：滑动窗口+哈希表

fast指针不断右移，当窗口内覆盖子串后，记录下标位置并左指针不断左移，缩圈。

**代码**：

```java
class Solution {
    public String minWindow(String s, String t) {
        int slow=0;
        int len=Integer.MAX_VALUE;
        int s1=-1;
        int s2=-1;
        HashMap<Character, Integer> tMap = new HashMap<>();
        HashMap<Character, Integer> sMap = new HashMap<>();
        char[] schars = s.toCharArray();
        char[] tchars = t.toCharArray();
        for (int i = 0; i < tchars.length; i++) {
            tMap.put(tchars[i],tMap.getOrDefault(tchars[i],0)+1);
        }
        for (int fast = 0; fast < schars.length; fast++) {
            sMap.put(schars[fast],sMap.getOrDefault(schars[fast],0)+1);
            while(check(sMap,tMap) && slow<=fast){
                if (fast-slow+1 < len){
                    s1=slow;
                    s2=fast;
                    len=fast-slow+1;
                }
                sMap.put(schars[slow],sMap.get(schars[slow])-1);
                slow++;
            }
        }
        if (s1 == -1){
            return "";
        }else{
            return s.substring(s1,s2+1);
        }
    }
    // 检查是否满足 t 的要求，-1 表示不满足，1 表示满足
    public boolean check(HashMap<Character, Integer> sMap, HashMap<Character, Integer> tMap) {
        for (Character character : tMap.keySet()) {
            if (sMap.getOrDefault(character,0) < tMap.get(character)){
                return false;
            }
        }
        return true;
    }

}
```
