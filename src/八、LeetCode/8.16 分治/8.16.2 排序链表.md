# 8.16.2 排序链表

leetcode.148

## 1、题目

给你链表的头结点 `head` ，请将其按 **升序** 排列并返回 **排序后的链表** 。



## 2、分析

链表不能随机访问，所以**不能用快排**，最优就是：

**归并排序（Merge Sort） + 快慢指针找中点**

1. 为什么用归并？

- 数组快排：靠下标随机访问，链表做不到
- 归并只需要：**切两半 → 递归排序左右 → 合并两个有序链表**
- 天然适合链表，且稳定、O (n log n)

2. 三步法

1. **找中点**：快慢指针，快指针走两步，慢走一步，最后慢指针就是中点
2. **切分**：从中间断开，分成左、右两个链表
3. **递归排序左右**，再**合并两个有序链表**[8.8.3 合并两个有序链表](https://ranqingisfine.github.io/myBlog/%E5%85%AB%E3%80%81LeetCode/8.8%20%E9%93%BE%E8%A1%A8/8.8.3%20%E5%90%88%E5%B9%B6%E4%B8%A4%E4%B8%AA%E6%9C%89%E5%BA%8F%E9%93%BE%E8%A1%A8.html)

```
例如： 2 4 3 1
先切分，然后递归排序

    1234
   /    \
  24    13
 / \    / \
2   4  3   1
```



## 3、代码

```java
class Solution {
    public ListNode sortList(ListNode head) {
        // 递归出口：空或只有一个节点，已经有序
        if (head == null || head.next == null) {
            return head;
        }

        // 1. 快慢指针找中点
        ListNode slow = head;
        ListNode fast = head.next;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
        }

        // 2. 切分左右两半
        ListNode rightHead = slow.next;
        slow.next = null; // 断开

        // 3. 递归排序左右
        ListNode left = sortList(head);
        ListNode right = sortList(rightHead);

        // 4. 合并两个有序链表
        return merge(left, right);
    }

    // 合并两个有序链表（LeetCode21）
    private ListNode merge(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0);
        ListNode cur = dummy;

        while (l1 != null && l2 != null) {
            if (l1.val < l2.val) {
                cur.next = l1;
                l1 = l1.next;
            } else {
                cur.next = l2;
                l2 = l2.next;
            }
            cur = cur.next;
        }

        // 剩余部分接上
        cur.next = l1 != null ? l1 : l2;
        return dummy.next;
    }
}
```
