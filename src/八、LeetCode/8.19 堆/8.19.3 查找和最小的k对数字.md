# 8.19.3 查找和最小的k对数字

leetcode.373

## 1、题目

给定两个以 **非递减顺序排列** 的整数数组 `nums1` 和 `nums2` , 以及一个整数 `k` 。

定义一对值 `(u,v)`，其中第一个元素来自 `nums1`，第二个元素来自 `nums2` 。

请找到和最小的 `k` 个数对 `(u1,v1)`, ` (u2,v2)` ...  `(uk,vk)` 。

**示例 1:**

```
输入: nums1 = [1,7,11], nums2 = [2,4,6], k = 3
输出: [[1,2],[1,4],[1,6]]
解释: 返回序列中的前 3 对数：
     [1,2],[1,4],[1,6],[7,2],[7,4],[11,2],[7,6],[11,4],[11,6]
```



## 2、分析

例子：

```
nums1 = [1,7,11], nums2 = [2,4,6], k=3
初始堆：{1+2=3, 0, 0}
第 1 次弹出：(1,2) 和 = 3
加入：
(1,0) → 7+2=9
(0,1) → 1+4=5
堆现在：{5, 9}
第 2 次弹出：(1,4) 和 = 5
加入：
(0,2) →1+6=7
堆现在：{7,9}
第 3 次弹出：(1,6) 和 = 7
前 3 对就是：[[1,2],[1,4],[1,6]]
完美正确。
```

- 两个数组都是**升序**→ 最小的一定是 `(nums1[0], nums2[0])`

- 用一个**最小堆**，堆里存：`{和, i, j}`，堆顶永远是**当前和最小的数对**

- 循环 k 次：
  - 弹出堆顶（最小和），加入结果



## 3、代码

```java
class Solution {
    public List<List<Integer>> kSmallestPairs(int[] nums1, int[] nums2, int k) {
        List<List<Integer>> res = new ArrayList<>();
        PriorityQueue<int[]> heap = new PriorityQueue<>((a, b) -> (nums1[a[0]] + nums2[a[1]]) - (nums1[b[0]] + nums2[b[1]]));
        
        if (nums1.length == 0 || nums2.length == 0) return res;
        heap.offer(new int[]{0, 0});  // 只存坐标，不存和！
        
        while (!heap.isEmpty() && k-- > 0) {
            int[] cur = heap.poll();
            int i = cur[0], j = cur[1];
            
            res.add(List.of(nums1[i], nums2[j]));
            
            // ✅ 只往下走第一列（避免重复）
            if (j == 0 && i + 1 < nums1.length) {
                heap.offer(new int[]{i + 1, j});
            }
            
            // ✅ 往右走（安全）
            if (j + 1 < nums2.length) {
                heap.offer(new int[]{i, j + 1});
            }
        }
        return res;
    }
}
```
