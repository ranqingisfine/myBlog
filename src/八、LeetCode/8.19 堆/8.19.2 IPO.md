# 8.19.2 IPO

leetcode.502

## 1、题目

假设 力扣（LeetCode）即将开始 **IPO** 。为了以更高的价格将股票卖给风险投资公司，力扣 希望在 IPO 之前开展一些项目以增加其资本。 由于资源有限，它只能在 IPO 之前完成最多 `k` 个不同的项目。帮助 力扣 设计完成最多 `k` 个不同项目后得到最大总资本的方式。

给你 `n` 个项目。对于每个项目 `i` ，它都有一个纯利润 `profits[i]` ，和启动该项目需要的最小资本 `capital[i]` 。

最初，你的资本为 `w` 。当你完成一个项目时，你将获得纯利润，且利润将被添加到你的总资本中。

总而言之，从给定项目中选择 **最多** `k` 个不同项目的列表，以 **最大化最终资本** ，并输出最终可获得的最多资本。

答案保证在 32 位有符号整数范围内。

**示例 1：**

```
输入：k = 2, w = 0, profits = [1,2,3], capital = [0,1,1]
输出：4
解释：
由于你的初始资本为 0，你仅可以从 0 号项目开始。
在完成后，你将获得 1 的利润，你的总资本将变为 1。
此时你可以选择开始 1 号或 2 号项目。
由于你最多可以选择两个项目，所以你需要完成 2 号项目以获得最大的资本。
因此，输出最后最大化的资本，为 0 + 1 + 3 = 4。
```

题目含义：

```
你一开始有 初始资本 w你最多能做 k 个项目
每个项目有两个值：
capital[i]：启动资金（你钱 ≥ 它才能做）
profit[i]：做完赚的钱（直接加到你的资本里）
规则：每次只能在你钱够做的项目里，选最赚钱的那个做。做完钱变多，再选下一个。做满 k 次，或没项目可做就停。
```



## 2、分析

```
例如：
k = 2（最多做 2 个项目）w = 2（初始钱：2）
profits = [3, 1, 2, 5]capital = [1, 3, 2, 4]
把项目写成人话：
项目 A：启动 1，赚 3
项目 B：启动 3，赚 1
项目 C：启动 2，赚 2
项目 D：启动 4，赚 5
----------------------------
首先：按照启动资金从小到大排序
排序后：
A：资本 1，利润 3
C：资本 2，利润 2
B：资本 3，利润 1
D：资本 4，利润 5
-------------------------
开始做项目（一共做 k=2 轮）
第 1 轮（第 1 次选项目）
当前钱：w = 2
我们把所有钱够做的项目丢进「大根堆」（最赚钱的放堆顶）
钱够（≤2）的项目：
A（1 ≤ 2 ✔）
C（2 ≤ 2 ✔）
堆里现在：3、2堆顶是 3（最大利润）
选它！w = 2 + 3 = 5
现在做了 1 个，还剩 1 次。
-------------------------
第 2 轮（第 2 次选项目）
当前钱：w = 5
继续把新够钱做的项目加入堆：B（3 ≤5 ✔）D（4 ≤5 ✔）
堆里现在剩下：2、1、5堆顶是 5（最大利润）
选它！w = 5 + 5 = 10
两次做完，结束。
```

总结：

- 项目按**启动资金排序**

- 用一个指针，把**当前钱够做的**全扔进**大根堆（利润最大优先）**

- 每次取堆顶（最赚），钱变多

- 再加入新够钱的项目

- 重复 k 次



## 3、代码

```java
class Solution {
    public int findMaximizedCapital(int k, int w, int[] profits, int[] capital) {
        int n=profits.length;
        int[][] projects=new int[n][2];
        for (int i = 0; i < n; i++) {
            projects[i][0]=capital[i];
            projects[i][1]=profits[i];
        }
        //将二维数组按照profits升序排列
        Arrays.sort(projects,(a,b)->a[0]-b[0]);
        PriorityQueue<Integer> heap=new PriorityQueue<>((a,b)->b-a);
        //遍历项目个数，每次要选择收益最大的项目
        //项目下标
        int idx=0;
        for (int i = 0; i < k; i++) {
            while (idx<n && projects[idx][0]<=w){
                heap.add(projects[idx][1]);
                idx++;
            }
            if (heap.isEmpty()){
                break;
            }
            w+=heap.poll();
        }
        return w;
    }
}
```
