# 8.6.1 汇总区间

Leetcode.228

**题目**：

给定一个  **无重复元素** 的 **有序** 整数数组 `nums` 。

区间 `[a,b]` 是从 `a` 到 `b`（包含）的所有整数的集合。

返回 ***恰好覆盖数组中所有数字** 的 **最小有序** 区间范围列表* 。也就是说，`nums` 的每个元素都恰好被某个区间范围所覆盖，并且不存在属于某个区间但不属于 `nums` 的数字 `x` 。

```
示例 1：
输入：nums = [0,1,2,4,5,7]
输出：["0->2","4->5","7"]
解释：区间范围是：
[0,2] --> "0->2"
[4,5] --> "4->5"
[7,7] --> "7"
```



**分析**：

1. 外层循环：确定每个区间的起始位置 `low`；
2. 内层循环：向后遍历找到当前区间的结束位置 `high`（直到不满足 “连续 + 1” 的条件）；
3. 根据 `low` 和 `high` 的关系拼接区间字符串（连续则拼 `low->high`，否则只拼 `low`）；
4. 重复上述步骤，直到遍历完整个数组。



**代码**：

```java
class Solution {
    public List<String> summaryRanges(int[] nums) {
        List<String> list=new ArrayList<>();
        int i=0;
        int n=nums.length;
        while(i<n){
            int low=i;
            i++;
            while(i<n && nums[i-1]+1==nums[i]){
                i++;
            }
            int high=i-1;
            StringBuilder temp=new StringBuilder();
            temp.append(nums[low]);
            if(low<high){
                temp.append("->");
                temp.append(nums[high]);
            }
            list.add(temp.toString());
        }
        return list;
    }
}
```
