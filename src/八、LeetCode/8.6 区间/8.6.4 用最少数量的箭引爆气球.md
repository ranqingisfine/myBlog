# 8.6.4 用最少数量的箭引爆气球

leetCode.452

**题目**：

有一些球形气球贴在一堵用 XY 平面表示的墙面上。墙面上的气球记录在整数数组 `points` ，其中`points[i] = [xstart, xend]` 表示水平直径在 `xstart` 和 `xend`之间的气球。你不知道气球的确切 y 坐标。

一支弓箭可以沿着 x 轴从不同点 **完全垂直** 地射出。在坐标 `x` 处射出一支箭，若有一个气球的直径的开始和结束坐标为 `xstart`，`xend`， 且满足  `xstart ≤ x ≤ xend`，则该气球会被 **引爆** 。可以射出的弓箭的数量 **没有限制** 。 弓箭一旦被射出之后，可以无限地前进。

给你一个数组 `points` ，*返回引爆所有气球所必须射出的 **最小** 弓箭数* 。

```
示例 1：

输入：points = [[10,16],[2,8],[1,6],[7,12]]
输出：2
解释：气球可以用2支箭来爆破:
-在x = 6处射出箭，击破气球[2,8]和[1,6]。
-在x = 11处发射箭，击破气球[10,16]和[7,12]。
```



**分析**：

- 按气球右边界升序排序（射在右边界能覆盖最多后续气球）；

- 初始箭数为 1，射在第一个气球的右边界；

- 遍历后续气球，若气球左边界超过箭的位置，新增箭并更新箭的位置为当前气球右边界。



**代码**：

```java
class Solution {
    public int findMinArrowShots(int[][] points) {
        // 判空
        if (points.length == 0) {
            return 0;
        }
        // 排序：用 Integer.compare 避免数值溢出，按区间右边界升序，Integer.compare无溢出，返回 - 1/0/1，稳定
        Arrays.sort(points, (o1, o2) -> Integer.compare(o1[1], o2[1]));
        // 处理
        int arrow = 1;
        int pos = points[0][1]; // 第一支箭射在第一个气球的右边界（贪心最优）
        for (int[] point : points) {
            // 核心判断：气球左边界 > 箭的位置 → 必须新增箭
            // （等于时，箭能引爆，无需新增）
            if (point[0] > pos) {
                arrow++;
                pos = point[1]; // 新箭射在当前气球的右边界
            }
        }
        return arrow;
    }
}
```
