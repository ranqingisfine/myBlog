# 8.6.2 合并区间

leetCode.56

**题目**：以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [starti, endi]` 。请你合并所有重叠的区间，并返回 *一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间* 。

```
示例 1：
输入：intervals = [[1,3],[2,6],[8,10],[15,18]]
输出：[[1,6],[8,10],[15,18]]
解释：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].
```



**分析**：

- 排序：将所有区间按「起始位置 start 升序」排列（确保重叠区间相邻）；

- 贪心合并：遍历排序后的区间，维护一个 “当前合并区间”，若当前区间与 “当前合并区间” 重叠，则合并（更新 end 为两者的最大值）；若不重叠，则将 “当前合并区间” 加入结果，更新为新的区间；

- 遍历结束后，将最后一个 “当前合并区间” 加入结果。

（此题可以当成API使用）

**代码**：

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        // 边界条件：空数组/单区间直接返回
        if (intervals == null || intervals.length <= 1) {
            return intervals;
        }
        
        // 步骤1：按区间起始位置升序排序
        Arrays.sort(intervals, (a, b) -> a[0] - b[0]);
        
        // 步骤2：贪心合并区间
        List<int[]> merged = new ArrayList<>();
        // 初始化当前合并区间为第一个区间
        int[] current = intervals[0];
        merged.add(current);
        
        for (int[] interval : intervals) {
            int currentEnd = current[1]; // 当前合并区间的结束位置
            int nextStart = interval[0]; // 下一个区间的起始位置
            int nextEnd = interval[1];   // 下一个区间的结束位置
            
            if (nextStart <= currentEnd) {
                // 重叠：合并区间，更新当前区间的结束位置为两者最大值
                current[1] = Math.max(currentEnd, nextEnd);
            } else {
                // 不重叠：将下一个区间作为新的当前区间，加入结果
                current = interval;
                merged.add(current);
            }
        }
        
        // 转换为数组返回
        return merged.toArray(new int[merged.size()][]);
    }
}
```
