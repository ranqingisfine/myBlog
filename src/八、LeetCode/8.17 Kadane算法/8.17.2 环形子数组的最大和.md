# 8.17.2 环形子数组的最大和

leetcode.918

## 1、题目

给定一个长度为 `n` 的**环形整数数组** `nums` ，返回 *`nums` 的非空 **子数组** 的最大可能和* 。

**环形数组** 意味着数组的末端将会与开头相连呈环状。形式上， `nums[i]` 的下一个元素是 `nums[(i + 1) % n]` ， `nums[i]` 的前一个元素是 `nums[(i - 1 + n) % n]` 。

**子数组** 最多只能包含固定缓冲区 `nums` 中的每个元素一次。形式上，对于子数组 `nums[i], nums[i + 1], ..., nums[j]` ，不存在 `i <= k1, k2 <= j` 其中 `k1 % n == k2 % n` 。

## 2、分析

环形数组的最大子数组和，只有**两种可能**：

1. 最大和在数组中间，不跨首尾

   

   → 就是普通 53 题：

   maxSub

2. 最大和跨了首尾

   （左边一段 + 右边一段）

   → 等价于：

   总和 total - 最小子数组和 minSub

最终答案就是：

**ans = max(maxSub, total - minSub)**

最大和跨了首尾为等价于total-最小数组和minSub?

```
例如：
数组：[5, -3, 5]
A = 5
B = -3 （最小子数组）
C = 5
total = 5 + (-3) + 5 = 7minSub = -3
total − minSub = 7 − (-3) = 10
而跨首尾最大子数组就是：5 + 5 = 10
```

注意：如果当前找到的最大子数组和是负数，就直接返回这个负数。

如果数组**全是负数**，比如：`[-5, -3, -2, -1]`

- 最大的子数组就是**单独选最大的那个负数**：`-1`

这时候 `maxSub` 就是 `-1`（负数）

所以直接返回它就行

## 3、代码

```java
class Solution {
    public int maxSubarraySumCircular(int[] nums) {
        int curMax = nums[0], maxSub = nums[0];
        int curMin = nums[0], minSub = nums[0];
        int total = nums[0];

        for (int i = 1; i < nums.length; i++) {
            // 求最大子数组和
            curMax = Math.max(nums[i], curMax + nums[i]);
            maxSub = Math.max(maxSub, curMax);

            // 求最小子数组和
            curMin = Math.min(nums[i], curMin + nums[i]);
            minSub = Math.min(minSub, curMin);

            total += nums[i];
        }

        // 全是负数的情况
        if (maxSub < 0) return maxSub;

        return Math.max(maxSub, total - minSub);
    }
}
```
